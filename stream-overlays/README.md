# 配信オーバーレイ — 藤 優真

OBS Studio で使う配信画面パーツ一式です。
すべて HTML/CSS で作られており、OBS の「ブラウザソース」として読み込みます。

---

## ファイル一覧

| ファイル | 用途 | 説明 |
|---|---|---|
| `waiting.html` | 待機画面 | 配信開始前に表示。名前 + 「まもなくはじまります」 |
| `main-talk.html` | メイン画面（雑談） | 立ち絵スペース左側 + ネームプレート + トピック表示 |
| `main-reading.html` | メイン画面（朗読/シチュボ） | 没入感重視。作品タイトル + テキスト表示エリア |
| `ending.html` | 終了画面 | 感謝メッセージ + SNS情報 |

---

## OBS への設定手順

### Step 1: シーンを作成する

OBS で以下の 4 つのシーンを作成：

1. **【待機】配信前**
2. **【メイン】雑談**
3. **【メイン】朗読・シチュボ**
4. **【終了】エンディング**

### Step 2: 各シーンにブラウザソースを追加する

各シーンで「ソースを追加」→「ブラウザ」を選択し、以下の設定を行う：

**共通設定：**
- 幅: `1920`
- 高さ: `1080`
- 「ローカルファイル」にチェック
- 該当する HTML ファイルのパスを指定

**ファイルパスの例（環境に合わせて変更）：**
```
C:\Users\MasaruShindo\work2\001_声優・音声\VoiceActorLaboratory\stream-overlays\waiting.html
```

### Step 3: 各シーンにソースを追加する

#### 【待機】配信前
| ソース | 種類 | 備考 |
|---|---|---|
| waiting.html | ブラウザ | 全画面 (1920x1080) |
| BGM | メディアソース | 待機用BGM（ループ再生ON） |

#### 【メイン】雑談
| ソース | 種類 | 備考 |
|---|---|---|
| main-talk.html | ブラウザ | 全画面オーバーレイ（背景透過） |
| 立ち絵 | 画像 | 左側 0〜500px 内に配置 |
| マイク | 音声入力キャプチャ | — |
| BGM | メディアソース | トーク用BGM（音量控えめ） |
| コメント表示 | ブラウザ ※任意 | わんコメ等。右下 (380x500) に配置 |

#### 【メイン】朗読・シチュボ
| ソース | 種類 | 備考 |
|---|---|---|
| main-reading.html | ブラウザ | 全画面（背景色あり） |
| 立ち絵 | 画像 | 左側 0〜420px 内に配置 |
| マイク | 音声入力キャプチャ | — |
| BGM | メディアソース | 雰囲気に合った環境音・BGM |

#### 【終了】エンディング
| ソース | 種類 | 備考 |
|---|---|---|
| ending.html | ブラウザ | 全画面 (1920x1080) |
| BGM | メディアソース | ED用BGM |

### Step 4: ブラウザソースの背景透過

`main-talk.html` は背景が `transparent` に設定されています。
OBS のブラウザソース設定で **「カスタムCSS」欄** に以下が入っていることを確認：

```css
body { background-color: rgba(0, 0, 0, 0); margin: 0px auto; overflow: hidden; }
```

これにより、オーバーレイの下に別の映像（立ち絵、背景画像等）を重ねられます。

## カスタマイズ

### 名前を変更する場合
各 HTML ファイル内の以下を検索・置換：
- `藤 優真` → 表示したい名前
- `Fuji Yuma` / `FUJI YUMA` → 英語表記

### トピック表示を変更する場合
`main-talk.html` の以下の部分を編集：
```html
<div class="topic-text">フリートーク</div>
```

### 作品タイトルを変更する場合
`main-reading.html` の以下の部分を編集：
```html
<div class="work-title">作品タイトル</div>
```

### SNS情報を変更する場合
`ending.html` の `.social-area` 内を編集。

### カラーテーマを変更する場合
各ファイルで共通して使っている主要カラー：
- **ゴールド**: `#d4af37` / `rgba(212, 175, 55, ...)` — アクセントカラー
- **ネイビー**: `#0a0e1a` / `#141b2d` — 背景
- **テキスト**: `#e8e6e3` — メインテキスト

## 推奨 OBS マイク設定（声優配信向け）

声を最大限活かすためのフィルタ設定例：

| フィルタ | 設定例 | 用途 |
|---|---|---|
| ノイズ抑制 | RNNoise | 環境ノイズを除去 |
| ノイズゲート | 開放: -32dB / 閉鎖: -36dB | 無音時のノイズカット |
| コンプレッサー | 比率 3:1 / 閾値 -18dB / アタック 6ms / リリース 60ms | 声量の均一化 |
| リミッター | -3dB | 音割れ防止 |

※ 順番は上から下の順でフィルタを追加すること

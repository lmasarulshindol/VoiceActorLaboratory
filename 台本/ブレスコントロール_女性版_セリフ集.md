# ブレスコントロール 女性版 セリフ集
# 息継ぎ・ロングブレス・ウィスパーの実践練習

> **目的**: 女性声優がブレスコントロール（息継ぎの技術）を体系的に身につけるための実践セリフ集
> **対象**: 女性声優志望者 / 現役女性声優のスキルアップ / 女性ナレーター
> **各セリフ目安**: 15〜30秒（100〜200文字）
> **使い方**: 各セリフの前にブレス基礎練習（腹式呼吸の確認）→ セリフ演技 → 録音して息継ぎ位置・ブレスノイズを確認

---

## ■ ブレスコントロール 基礎知識

### ブレスの種類と特性比較

| ブレスの種類 | 呼吸の仕組み | 息の持続力 | 声の安定感 | 適した場面 |
|------------|-----------|----------|----------|----------|
| **腹式呼吸** | 横隔膜を下げて肺底部を広げる | 長い（安定した呼気） | 非常に高い | ロングブレス、ナレーション、安定した長台詞 |
| **胸式呼吸** | 肋骨を広げて肺上部を使う | 短い（浅い呼気） | 低い（声が揺れやすい） | 緊迫シーンの短い叫び、息切れ演技 |
| **混合式** | 腹式＋胸式を意識的にブレンド | 中〜長い（柔軟） | 高い | 緩急の切り替え、感情的な長台詞 |

### ブレス技術別の使い分け

| ブレス技術 | 技術の概要 | 有効なシーン |
|----------|----------|------------|
| **ロングブレス** | 一息で長い台詞を読み切る | 舞台系の独白、告白シーン、感情が途切れてはいけない場面 |
| **ブレスマーク** | 息継ぎ位置を最適化し自然なフレージングを作る | ニュース、ナレーション、天気予報、情報量の多い台詞 |
| **ウィスパーブレス** | 息多めの繊細な発声で空気感を演出 | ASMR、夜の語り、乙女ゲーム、心理描写 |
| **緩急ブレス** | 叫びと囁き、激と静の切り替え | 変身シーン、感情の爆発と抑制の往復 |
| **無声ブレス** | 息継ぎ音をマイクに拾わせない技術 | ラジオドラマ、ナレーション収録、ASMR系収録 |

### 女性特有のポイント

- 女性は男性より平均肺活量が20〜30%少ない → ロングブレスでは「息の節約」が男性以上に重要。呼気圧を低めに設定し、最小限の空気で最大限の声を維持する技術が求められる
- 高音域での発声は声帯の振動が速く、息の消費量が大きい → 高音キャラ（魔法少女、アイドル等）を演じる際は、中低音キャラの1.5倍の呼気を消費する前提で息を設計する
- 女性の声帯は薄く柔軟なため、ウィスパーボイスとの親和性が高い → ただし「息っぽさ」に頼りすぎると声帯が疲労しやすい。収録の後半にウィスパーパートを配置しないよう注意
- 生理周期によって声帯のコンディションが変動する → むくみが出やすい時期はブレスが浅くなりがち。体調に応じてブレスの深さを意識的に調整すること
- 胸式呼吸が癖になっている人が多い → 肩が上がる呼吸はNG。腹式呼吸を徹底し、横隔膜を「下に押す」感覚を体で覚える

### ブレスコントロールのコツ

- **息の「貯金」を作る**: 台詞の開始前に肺の80%まで吸い、20%の余力を残す。100%まで吸うと体が硬くなり声が固まる
- **呼気圧の一定化**: ロウソクの炎を揺らさず吹き続けるイメージ。息の量ではなく「圧」をコントロールする
- **ブレスポイントの設計**: 文の意味の切れ目で吸い、感情の流れを断ち切らない位置を選ぶ
- **鼻吸い vs 口吸い**: 鼻吸いはノイズが少なく深い吸気。口吸いは速いが音が出やすい。シーンに応じて使い分ける
- **支え（サポート）の維持**: 息を吐き終わっても腹筋の支えを緩めない。支えが抜けた瞬間、声がブレる
- **高音域での省エネ意識**: 女性の高音域は息の消費が激しい。高い声を出す場面ほど、呼気圧を「絞る」意識で。力任せに息を吐くと声が裏返る

# 【ロングブレスパート】

> **技術の特徴**: 一息で長い台詞を途切れなく読み切る。感情の流れを「息の流れ」に乗せて持続させる。
> **ウォームアップ**: 腹式呼吸で「すー」と息を吐く → 15秒 → 20秒 → 25秒と段階的に延ばす → 息の上に「あー」と声を乗せる
> **女性向け注意**: 肺活量の少なさを「呼気の精密さ」で補う。息を大量に使わなくても声は維持できる。呼気圧を一定に保ち、最後の一語まで声の芯を残すこと。

## ロングブレス① 情熱的なヒロインの告白（恋愛ゲーム・アニメ）

> **想定媒体**: 恋愛ゲーム / アニメ / シチュエーションCD
> **キャラクター**: 17歳の高校二年生。ずっと想いを隠してきた幼馴染への告白シーン。放課後の教室、二人きり。言い出す勇気がなかなか出なかったけれど、卒業が近づく今、もう後がない——溢れ出す感情を一息に乗せて、途切れることなく伝え切る。
> **演技方針**: 台詞の開始前に深く吸い、一息で語り切る意識を持つ。途中で息が足りなくなっても「絞り出す」のではなく、呼気圧を下げて持続させる。声量のピークは「好きだよ」の直前に設計し、告白の頂点に向けて息の配分を逆算すること。感情が乗るほど息を消費しやすいが、「想いの強さ」と「息の配分」は別物。

（夕陽が差す教室。窓際に立つ相手の背中に向かって）

あのね、ずっと言えなかったことがあるの——小学校の帰り道、毎日一緒に歩いてくれたこと、中学の時、文化祭で私が泣いたら何も聞かずにそばにいてくれたこと、高校に入って少し距離ができても、朝、廊下ですれ違うたびに私の名前を呼んでくれたこと——全部、全部ずっと宝物だった。だから今、この声が届くうちに言わせて——私、あなたのことが好きだよ。

### ブレスポイント
- 「言えなかったことがあるの——」：ダッシュの位置で息を切りたくなるが、ここは呼気圧を落として耐える。感情の接続を息で途切れさせない
- 「宝物だった。」：ここまでの回想パートで息の7割を使う設計。前半は呼気圧を控えめにし、息を温存しながら語る
- 「好きだよ。」：最後の告白に残りの息を全て注ぐ。語尾が消えたら告白は届かない——腹筋の支えを最後まで維持
- **全体**: 約200文字を一息で読み切るロングブレスの極限トレーニング。女性は肺活量が少ない分、呼気の「精密な配分」が鍵。最初は途中で区切っても良いが、最終目標は一息

# 【ブレスマークパート】

> **技術の特徴**: 息継ぎの位置を最適化し、自然なフレージング（文のまとまり）を作る。情報を整理して届ける技術。
> **ウォームアップ**: 新聞の社説を音読し、句読点ごとに「鼻から吸う」練習 → 吸気の音が出ないよう意識
> **女性向け注意**: 一息が短い分、ブレスマークの設計がシビアになる。「吸える場所」と「吸うべき場所」を一致させる技術が重要。

## ブレスマーク① 女性キャスターの天気予報（情報番組）

> **想定媒体**: テレビの朝の情報番組 / 天気予報コーナー
> **キャラクター**: 20代後半の女性気象キャスター。明るく親しみやすいが、情報の正確さは妥協しない。視聴者が一度聴いただけで今日の天気を把握できる「聴きやすさ」が求められる。笑顔を声に乗せつつ、数字と地名は一切のあいまいさなく伝える。
> **演技方針**: ブレスマーク（/）の位置で素早く鼻から吸い、フレーズごとの「意味のかたまり」を声で区切る。吸気音をマイクに載せず、息継ぎの存在を感じさせないスムーズな呼吸が求められる。各フレーズを等しい呼気圧で読み、語尾が落ちないよう注意。明るいトーンと正確な情報伝達を両立させる。

続いて、/ お天気です。/ 今日の関東地方は / 朝から広く晴れて、/ 絶好のお出かけ日和となりそうです。// 最高気温は / 東京で二十四度、/ 横浜で二十三度、/ 前橋では二十六度と / 四月としてはやや高めの予想です。/ お出かけの際は / 日傘があると安心ですね。// ただし、/ 夕方以降は / 西から雨雲が近づく見込みで、/ 夜遅くには / 広い範囲で雨となるでしょう。/ 折りたたみ傘を / お忘れなく。

### ブレスポイント
- 「/」の位置：鼻から0.3秒で吸う。吸気音がマイクに入らないよう、鼻腔を広げてゆっくり空気を通す
- 「//」の位置：やや長めの間（0.5秒）。話題の転換を示すブレス。ここでしっかり肺に空気を補充する
- 「二十四度」「二十三度」「二十六度」：数字の前後で微細なブレスを入れ、数字を際立たせる。天気予報は数字が命
- **全体**: ブレスマークは「聴く人のための息継ぎ」。自分が苦しいから吸うのではなく、情報の区切りとして設計する。女性は一息が短い分、ブレスポイントの数が男性より多くなっても自然。大切なのはブレスの「等間隔感」と「無音性」

# 【ウィスパーボイスパート】

> **技術の特徴**: 息を多めに含んだ繊細な発声。声帯を完全に閉じず、空気を混ぜることで「囁き」の質感を作る。
> **ウォームアップ**: 「はぁー」と温かい息を吐く → 息の上に微かな声を乗せる → 声と息の比率を「3:7」→「5:5」→「7:3」と変化させる
> **女性向け注意**: 女性の薄く柔軟な声帯はウィスパーとの親和性が高い。しかし声帯を開きすぎると「ただの息」になり、閉じすぎると「普通の小声」になる。「息の中に声の粒が浮かんでいる」感覚を見つけること。

## ウィスパー① 乙女ゲーム系ヒロインの夜の独白（ボイスコンテンツ）

> **想定媒体**: 乙女ゲーム / シチュエーションCD / 睡眠導入ボイス
> **キャラクター**: 20歳の大学生。好きな人と電話で「おやすみ」を言った後、一人ベッドの上で、まだドキドキしている胸に手を当てながら、誰にも聞こえない声でぽつりと想いを零す。
> **演技方針**: 声帯を半開きにし、吐く息に声をそっと乗せる。「聞こえるか聞こえないか」のギリギリの音量で、しかし子音は明瞭に保つ。マイクとの距離は近め（10〜15cm）を想定。呼気圧を極限まで下げ、息が「絹糸のようにほどけていく」リズムを作る。

（暗い部屋。ベッドの上で仰向けになり、スマホを胸に抱えて）

……電話、切れちゃった。
……まだ耳に残ってる。「おやすみ」って言ってくれた声。
（寝返りを打って、枕に顔を埋めるように）
……ねぇ、聞こえてないよね。もう切れてるよね。
——好きだよ。……すっごく、好き。
……言えたらいいのに。いつか、ちゃんと声に出して。
（目を閉じて、消えるように）
……おやすみ。——今夜も、夢で逢えるかな。

### ブレスポイント
- 「……」の位置：ここで静かに鼻から吸う。吸気自体が「余韻の間」になる。口から吸うとウィスパーの空気感が壊れる
- 「好きだよ。」：ウィスパーの核心。声帯の閉鎖度をほんの少しだけ上げ、「好き」の一語だけ声の芯を通す。全体がウィスパーだからこそ、わずかな芯が際立つ
- 「夢で逢えるかな。」：語尾を完全にフェードアウトさせる。呼気圧を徐々にゼロへ近づけ、声が「消える」瞬間を作る
- **全体**: ウィスパーは「息のコントロール」が9割。声量ではなく「息の温度と速度」で表現する。女性のウィスパーは声帯が薄い分、息と声の境界が曖昧になりやすい——「声の粒」を見失わないよう、子音の輪郭だけは常にキープすること

# 【緩急ブレスパート】

> **技術の特徴**: 叫びから囁き、あるいは囁きから叫びへ。呼気圧の急激な切り替えで、感情のコントラストを表現する。
> **ウォームアップ**: 「はぁーっ！」（全力呼気）→ 即座に「……はぁ」（最小呼気）→ これを5回繰り返し、切り替え速度を上げる
> **女性向け注意**: 女性は高音域で叫ぶと息の消費が特に激しい。叫びのパートで息を使い切ると、静かなパートが成立しない。「7割の力で叫ぶ」意識が大切。また、叫びの後に声が裏返りやすいので、リセットブレスで声帯をニュートラルに戻す練習をすること。

## 緩急ブレス① 魔法少女の変身〜静かな決意（アニメ・ゲーム）

> **想定媒体**: アニメ / ゲーム / ドラマCD
> **キャラクター**: 14歳の中学生。魔法少女として大切な人を守るために戦う。変身シーンでは声を天に向けて解き放ち、変身後は静かに、しかし揺るぎない決意で敵と向き合う。華やかな叫びから、芯の通った囁きへ——感情の落差を息で描く。
> **演技方針**: 前半は混合式呼吸で呼気を一気に放出する「叫び」。叫びの後に0.5秒の「リセットブレス」を挟み、後半は呼気圧を十分の一に落とした「囁きに近い語り」へ移行する。叫びで息を使い切らず、後半の静かなパートのために呼気を残す配分が鍵。

（杖を天に掲げて、全力の叫び）

響け、星の鼓動！　——プリズム・トランスフォーメーション！！

（光が収まる。変身後、静寂の中で——）

……はぁ……。

（目の前の敵をまっすぐ見据えて、静かに）

……あなたが何を壊そうとしているか、私は知ってる。
あの子たちの笑顔。当たり前の毎日。……私が守りたい、全部。

（杖を構え直して、囁くように、けれど一切の迷いなく）

——だから、ここは通さない。

### ブレスポイント
- 「プリズム・トランスフォーメーション！！」：混合式呼吸で呼気を爆発させる。ただし全力の7割に抑え、後半のためにに息を残す。女性の高音叫びは特に息を消費するため、力任せにならないこと
- 「はぁ……。」：変身後の一息。ここでリセットブレスを入れ、叫びの喉の緊張を解放する。鼻から深く吸い直し、声帯をニュートラルに戻す
- 「全部。」：ウィスパーに近い呼気圧で、しかし声の芯は残す。叫びとの対比が大きいほど、静かな言葉の重みが増す
- **全体**: 叫び→静寂の切り替えで最も重要なのはリセットブレス。叫びの直後に鼻から深く一度吸い、声帯の緊張を解放してから静かなパートに入る。女性は声帯が繊細な分、この「リセット」を丁寧に行わないと声がかすれる

# 【無声ブレスパート】

> **技術の特徴**: 息継ぎの音をマイクに拾わせない技術。鼻から静かに、かつ深く吸うことで、リスナーにブレスの存在を意識させない。
> **ウォームアップ**: マイク（またはスマホ）を口元10cmに置き、鼻から吸気 → 録音して再生 → 吸気音が聞こえなくなるまで繰り返す
> **女性向け注意**: 女性は一息が短い分、吸気の回数が多くなる。回数が増えるほどブレスノイズのリスクも上がるため、「少ない吸気で最大限の空気を取り込む」効率が男性以上に求められる。

## 無声ブレス① ラジオドラマの語り手（ラジオ・配信・ASMR）

> **想定媒体**: ラジオドラマ / Podcast / ASMR系コンテンツ
> **キャラクター**: 20代後半の女性。ラジオドラマの一人称語り手。深夜帯の番組で、リスナーはイヤホンで聴いている。静かな声の中にブレスノイズが入れば、それは没入感を壊す致命的なミスになる。
> **演技方針**: 全てのブレスを鼻から無音で行う。吸気に使える時間は句読点の「間」の0.3〜0.5秒のみ。呼気圧を一定に保ち、フレーズの頭と末尾で声量差が出ないようにする。マイクとの距離は20〜25cmを想定し、吹かれ（ポップノイズ）にも注意。

あの日、桜が散り始めた午後だった。
駅の改札を出ると、三月の風がスカートの裾を揺らした。
見慣れたはずの坂道は、どこか違って見えた。
角を曲がれば、あの小さなパン屋がある——はずだった。
けれど、そこにあったのは、更地と、白いフェンスだけ。
（一拍の静寂）
——私はしばらく、その場に立ち尽くしていた。
ポケットの中のスマホが震えたけれど、取る気にはなれなかった。

### ブレスポイント
- 「午後だった。」：句点の後に0.5秒の間。この間に鼻から無音で吸う。鼻孔をわずかに広げ、ゆっくり空気を通す
- 「揺らした。」「見えた。」：句点ごとの短い間（0.3秒）で吸う「スナッチブレス」。女性は一息が短いため、男性より頻繁にスナッチブレスを挟む必要がある——その全てを無音で行う
- 「——私はしばらく」：ダッシュの前に0.5秒の間。ここは「間の演出」と「無声ブレス」を兼ねる最重要ポイント
- **全体**: 録音後に波形を確認し、ブレス位置にノイズが入っていないかチェックする。無声ブレスは「聴こえないこと」が成功の基準。一箇所でも「スッ」と音が入れば失敗。女性は吸気回数が多い分、全ての吸気を無音にする難易度が男性より高い——そこにこそ鍛える価値がある

# 【応用：複合ブレス技術の実践】

> **目的**: 複数のブレス技術を一つのセリフの中で組み合わせ、実践的な収録に対応できる応用力を鍛える。
> **女性向けポイント**: 技術の切り替え時に声質がブレやすい。各技術の移行点で「リセットブレス（鼻から静かに一度吸い直す）」を意識すること。高音域と低音域の行き来がある場合は、声帯への負担に特に注意。

## 応用① ロングブレス＋ウィスパー（手紙の朗読から心の声へ）

> **シチュエーション**: 遠くにいる大切な人からの手紙を声に出して読み、途中から声が手紙を離れ、自分自身の心の声に変わっていく。朗読のロングブレスから、ウィスパーの独白へ。
> **技術の流れ**: ロングブレス（手紙の朗読を一息で）→ リセットブレス → ウィスパー（心の声・独白）

（手紙を手に取り、声に出して読み始める——一息で）

「元気にしていますか、こちらはもう冬で、窓から見える山が真っ白になりました、あなたが好きだった焼き栗の屋台が今年も駅前に出ていて、食べるたびにあなたのことを思い出しています、また会える日を楽しみにしています」

（手紙を胸に当てて、目を閉じる。声がウィスパーに変わる）

……また会える日、か。
……私も、毎日思ってるよ。
（消え入りそうな声で）
……早く——会いたいな。

### ブレスポイント
- 手紙パート（約130文字）：一息で読み切る設計。手紙の文面は句読点なしで書かれており、息継ぎのきっかけがない——呼気圧を極限まで絞り、最小限の息で最大限の声を保つ
- 手紙パート終了後：最重要のリセットブレス。深く鼻から吸い、ロングブレスの呼気モードからウィスパーの繊細な呼気モードへ切り替える
- 「会いたいな。」：ウィスパーのフェードアウト。呼気をゼロに近づけて声が消える瞬間を作る。息が「溶けていく」感覚
- **ポイント**: ロングブレスで息を使い切ると、ウィスパーに移行できない。手紙パートは肺の6割で読み、4割をウィスパーに残す配分

## 応用② 緩急ブレス＋無声ブレス（ホラーゲーム実況風ナレーション）

> **シチュエーション**: ホラーゲームの実況風ナレーション。静かな語りから突然の叫び、そして恐怖に震える囁き。無声ブレスで没入感を維持しつつ、緩急の激しい感情を表現する。
> **技術の流れ**: 無声ブレス（静かな語り）→ 緩急ブレス（叫び）→ 無声ブレス（恐怖のウィスパー）

（淡々とした語り口で——ブレスノイズ一切なし）

廊下の奥、電気が点いていない部屋がある。
ドアは少しだけ開いている。隙間から冷たい風が流れてくる。
そっと、手をかけて——

（突然、何かが飛び出す。絶叫）

きゃあああっ！！

（息を荒げながら——しかし、すぐに声を潜めて）

……はぁっ……はぁ……。
……い、今の……何？　何だったの……？
（震える声で、ほとんど囁き）
……戻ろう。もう無理。——でも、後ろにも何かいる気がする……。
……どうすればいいの。

### ブレスポイント
- 前半のナレーション部分：全てのブレスを鼻から無音で。リスナーをイヤホンで聴いている前提——ブレスノイズは恐怖の前に没入感を壊す
- 「きゃあああっ！！」：呼気を一気に放出する叫び。ただし全力の7割で——直後の恐怖パートのために息を残す
- 叫び直後の「はぁっ……はぁ……」：荒い呼吸を「演技としてのブレス」に変換。ここだけは呼吸音を「聴かせる」——恐怖のリアリティ
- 後半のウィスパー部分：呼吸音以外のブレスは再び無声に戻す。「聴かせるブレス」と「消すブレス」の使い分けが、このセリフの核心
- **ポイント**: 無声ブレスと叫びは対極の技術。その急激な切り替えを一つの芝居の中で成立させる——実践的なブレスコントロールの総合力が問われる

## 応用③ ブレスマーク＋ロングブレス＋ウィスパー（結婚式のスピーチ）

> **シチュエーション**: 親友の結婚式でのスピーチ。前半は丁寧なブレスマークで祝辞を述べ、中盤で感極まってロングブレスの回想に入り、最後は涙をこらえたウィスパーで締める。
> **技術の流れ**: ブレスマーク（祝辞の情報整理）→ ロングブレス（思い出の語り）→ ウィスパー（涙をこらえた締め）

（マイクの前に立ち、落ち着いた声で——ブレスマーク意識）

本日は / おめでとうございます。/ 新婦の美咲とは / 中学一年からの付き合いで、/ かれこれ十五年になります。// 彼女がどんな人かを / 一言で言うなら / 「泣き虫だけど、誰よりも強い人」/ です。//

（感極まって、回想に入る——一息で）

高校の時、二人で受けた合唱コンクールで金賞逃して、帰り道にファミレスで三時間泣いて、でも次の日にはケロッとして「来年は絶対獲ろう」って笑ってた——あの笑顔が、私は大好きだった。

（涙をこらえながら、ウィスパーで）

……美咲、幸せにね。
——ずっと、ずっと応援してるから。

### ブレスポイント
- 前半のブレスマーク部分：「/」位置で鼻から0.3秒の無声ブレス。スピーチとして聴きやすいフレージングを設計する
- 中盤のロングブレス部分（約120文字）：一息で読み切る。感情が昂るほど息を消費するため、呼気圧を意識的に抑えて温存する
- 最後のウィスパー部分：涙をこらえる声。声帯を半開きにし、息の比率を上げる。「泣いている」のではなく「泣くのをこらえている」声——呼気の震えを演技に変換する
- **ポイント**: 三つの技術を自然な感情の流れの中で切り替える。技術が見えないのが理想——聴いている人には「感動的なスピーチ」としか映らないこと

# 【収録ガイド】

## 推奨収録順

### Step 1: ウォームアップ（ブレスの基礎確認）
1. **腹式呼吸の確認** → 仰向けに寝て腹部に手を当て、吸気で腹が膨らみ呼気で凹むことを確認。肩が上がらないよう注意
2. **呼気持続トレーニング** → 「すー」で15秒 → 20秒 → 25秒と延ばす。女性は25秒を安定して出せれば十分
3. **無声吸気の練習** → マイクの前で鼻から吸い、録音して吸気音が入っていないか確認
4. **リップロール** → 声帯のリラックス。女性は声帯が繊細なので、叫びのパートの前に必ず

### Step 2: 基礎技術パート（一つずつ丁寧に）
5. **無声ブレス①**（ラジオドラマの語り手）→ ブレスノイズゼロを目指す基礎トレーニング
6. **ブレスマーク①**（天気予報キャスター）→ 息継ぎ位置の最適化と無声吸気の併用
7. **ウィスパー①**（乙女ゲーム系ヒロイン）→ 息の比率を繊細にコントロールする

### Step 3: 高負荷パート（呼気量と切り替え）
8. **ロングブレス①**（ヒロインの告白）→ 一息で読み切る持続力
9. **緩急ブレス①**（魔法少女の変身）→ 叫びと囁きの急激な切り替え

### Step 4: 応用（複合技術）
10. **応用①**（手紙の朗読→心の声）→ ロングブレスとウィスパーの切り替え精度
11. **応用②**（ホラーゲーム実況風）→ 無声ブレスと緩急ブレスの複合
12. **応用③**（結婚式スピーチ）→ ブレスマーク＋ロングブレス＋ウィスパーの三重奏

## 録音チェックリスト

| チェック項目 | 確認方法 |
|------------|---------|
| 腹式呼吸ができているか | 腹部に手を当て、吸気で膨らみ呼気で凹む。肩が上がっていたら胸式に逃げているサイン |
| ブレスノイズが入っていないか | 録音の波形を確認し、フレーズ間に吸気の「スッ」音がないか。無声ブレスパートは特に厳密に |
| 語尾が落ちていないか | 各フレーズの最後の1〜2文字の音量が下がっていないか。呼気が足りていない証拠 |
| ロングブレスで声質が変わっていないか | 台詞の前半と後半で声の太さ・明瞭さに差がないか。後半が細くなるのは呼気圧低下のサイン |
| ウィスパーが「かすれ」になっていないか | 息っぽさの中に声の芯が残っているか。完全にかすれたらそれは声枯れであり、ウィスパーではない |
| 緩急の切り替えで声がブレていないか | 叫びの直後に声が裏返ったり震えたりしていないか。リセットブレスが機能しているか確認 |
| 高音の叫びで喉が締まっていないか | 叫びのパートで声が苦しそうに聞こえないか。喉ではなく腹腔から声を出せているか |
| ブレスマークが意味の切れ目と一致しているか | 息継ぎの位置が文の意味を断ち切っていないか。不自然な位置で吸っていないか |
| 全体のテンポが安定しているか | ブレスの長さ・深さがバラついていないか。特に緊張すると吸気が浅くなりテンポが崩れる |
| 声帯に疲労が溜まっていないか | 収録後半でウィスパーがかすれたり、高音が出にくくなっていないか。休憩を挟む判断も技術のうち |

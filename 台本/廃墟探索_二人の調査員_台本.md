# 廃墟探索 ― 二人の調査員

# 設定
廃墟となった山間の療養所を調査する二人の男。元新聞記者の「沢村」とフリーのカメラマン「神谷」。
ナレーションはドキュメンタリー風に状況を語る。

# トーン
緊張感のある低音。ナレーションは淡々と事実を述べる冷静さ。沢村は理性的で慎重。神谷は直感的で行動派。

# シーン1_到着

ナレーション: 県道から脇道に入り、未舗装の山道を20分ほど登った先。木々に覆われるようにして、その建物は建っていた。昭和38年に閉鎖された旧・北嶺療養所。築70年を超える鉄筋コンクリートの躯体は、苔と蔦に浸食されながらも、まだ崩壊を拒んでいるように見えた。

# シーン2_入口にて

沢村: ……ここか。写真で見るよりずっと大きいな。

神谷: ああ。3階建てだが、東棟はほぼ倒壊してる。入れるのは本館と西棟だけだ。

沢村: 足元に気をつけろ。床が抜けてる箇所がある。資料を見る限り、地下にボイラー室と倉庫がある。

神谷: 先に本館を回ろう。光が入る時間帯のうちに撮りたい。

# シーン3_本館内部

ナレーション: 玄関ホールに足を踏み入れると、空気が変わった。外の湿った緑の匂いが、埃と朽ちたコンクリートの乾いた匂いに取って代わる。天井の一部が崩落し、そこから差し込む午後の光が、受付カウンターの上に四角い影を落としていた。

沢村: ……まだ残ってるな。入院患者の記録簿だ。

神谷: 触るな。原本だろう。撮影だけにしておけ。

沢村: わかってる。だが見ろ、最終記入日が昭和38年2月17日。閉鎖日と一致する。突然だったんだな。

# シーン4_西棟の廊下

ナレーション: 西棟の廊下は、かつて患者たちが日光浴をしていたという南向きの回廊だった。ガラスはすべて割れ、代わりに風が自由に吹き抜けている。壁に残された色褪せたタイル画は、山と川を描いたもので、ここが療養の場であったことを静かに物語っていた。

神谷: ……沢村。聞こえるか、この音。

沢村: ……風だろう。廊下が煙突効果で鳴っている。

神谷: いや、違う。もっと……規則的だ。

沢村: ……地下だな。ボイラー室の配管が風で共鳴してるんだろう。確認するか？

神谷: ……行こう。ここまで来て引き返す理由もない。

# シーン5_帰路

ナレーション: 日没までに調査を終え、二人は山道を下った。沢村のノートには、建物の劣化状況と歴史的資料の所在が几帳面な字で記録されていた。神谷のカメラには、400枚を超える写真が収められていた。
ナレーション: この療養所が完全に解体されるまで、あと3年。記録できる時間は、もうそれほど残されていない。

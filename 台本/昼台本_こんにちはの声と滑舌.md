# 昼台本 — こんにちはの声と滑舌

# 設定
昼のひとときに読む台本。挨拶から始まり、昼の情景、姿勢と呼吸、午後への切り替え、滑舌の確認、感情を込めたセリフ、締めの言葉までを通しで収録。

# トーン
軽やかでにぎやかな昼の空気。力強く明瞭に。語尾までしっかり届ける。

# シーン1_挨拶と昼の情景

ナレーション: こんにちは。
ナレーション: 太陽が高く昇り、街がいちばん活気づく時間です。午前中の仕事や勉強を終えて、ほっとひと息つく頃でしょうか。窓から差し込む光が机の上を照らし、風がカーテンをやわらかく揺らします。昼の空気は、朝よりも軽く、夜よりもにぎやかです。

# シーン2_姿勢と呼吸

ナレーション: ここで一度、姿勢を整えます。背筋を伸ばし、あごを引き、視線をまっすぐ前へ。深く吸って、ゆっくり吐く。呼吸と声をつなげるように、「こんにちは」ともう一度、はっきり発音します。
朗読: こんにちは。
ナレーション: こ・ん・に・ち・は。母音を丸く、子音をくっきり。

# シーン3_午後への切り替え

ナレーション: さて、午後に向けて気持ちを切り替えましょう。
朗読: 準備は万端です。
朗読: 確認は完了しました。
朗読: これから本番です。
ナレーション: 短い文を、力強く、明瞭に。語尾を曖昧にせず、最後まで音を届けます。

# シーン4_滑舌の確認

ナレーション: 滑舌の確認です。
朗読: 生麦生米生卵。
朗読: 東京特許許可局。
ナレーション: 速さより正確さ。舌先の位置を意識し、歯の裏に軽く触れる感覚を確かめます。口を大きく開け、唇をしっかり使います。

# シーン5_感情を込めたセリフ

ナレーション: 次は少し感情を乗せます。
ナレーション: 嬉しそうに、「おいしいですね。」
朗読: おいしいですね。
ナレーション: 真剣に、「重要な点を説明します。」
朗読: 重要な点を説明します。
ナレーション: 落ち着いて、「どうぞご安心ください。」
朗読: どうぞご安心ください。
ナレーション: 同じ昼でも、状況はさまざま。声色を変えることで、場面が自然に伝わります。

# シーン6_締めくくり

ナレーション: 最後に、穏やかな調子で締めます。
ナレーション: 昼の時間は、流れの中間地点。焦らず、丁寧に、着実に。
ナレーション: 午後も、自分の声を信じて、一歩ずつ前へ進みます。

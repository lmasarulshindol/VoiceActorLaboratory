# 掛け合い台本 — ポンコツパーティーの休日大作戦

# 設定
アクセルの街。久しぶりのクエスト無しの休日。カズマが「たまにはのんびりしたい」と宣言するも、残る3人がそれぞれ好き勝手に行動した結果、屋敷は大混乱。アクアの宴会芸練習、めぐみんの爆裂魔法欲求不満、ダクネスの筋トレ＆変態妄想——収拾がつかなくなったカズマが全員を連れ出して結局クエストに出かける羽目に。ボケ×3・ツッコミ×1という地獄のような漫才構造で繰り広げる、このすばフルパーティーの日常コメディ。

# 登場人物（4名）
- カズマ / 佐藤和真（男性・普通〜やや低め）：ツッコミ兼指揮官。元引きこもりの冒険者。一人称「俺」。口癖「しょうがねえなぁ」「この駄女神」。冷静な皮肉と感情的な叫びを行き来する、唯一のツッコミ要員にしてパーティーの保護者。
- アクア（女性・明るく威勢がいい）：ボケ担当その1。水の女神。一人称は普段「あたし」、威厳モード時「私」。語尾「〜よ」「〜わよ」「〜なの」。嘲笑時「プークスクス」。自信満々→失敗→号泣のサイクルが早い構ってちゃん。酒好き・借金体質。
- めぐみん（女性・中性的なですます調）：ボケ担当その2。紅魔族のアークウィザード。13〜14歳。一人称「私」。普段は丁寧だが中二病スイッチONで大仰な口上。怒ると「ちがうわい！」。爆裂魔法への愛は絶対。1日1回撃たないと禁断症状が出る（嘘）。
- ダクネス / ダスティネス・フォード・ララティーナ（女性・芯のある低め）：ボケ担当その3。クルセイダー。18歳。一人称「私」。貴族令嬢だが極度のドM。丁寧口調だが興奮すると鼻息が荒くなり妄想が暴走。「むしろ望むところだ！」「ララティーナと呼ぶな！」が口癖。攻撃は全く当たらないが防御は鉄壁。仲間を守る騎士道精神は本物。

# トーン
ボケ×3・ツッコミ×1の変則漫才。カズマが1人に突っ込んでいる間に残り2人が新たなボケを量産し、カズマの処理能力を超えていく。ダクネスのドM発言にはカズマの「こいつもうダメだ」系のツッコミ、アクアには「この駄女神」系、めぐみんには「中二病やめろ」系と、3種類のツッコミスタイルを使い分ける。最終的に全員が一つの目標に向かう瞬間だけ、奇跡的にまとまる。

# 演技メモ

- **カズマ**: 唯一のツッコミとして、3方向から飛んでくるボケを捌く激務。ツッコミのバリエーションを使い分けること。アクアには「この駄女神！」と呆れ全開、めぐみんには「中二病やめろ」とツッコみつつ甘め、ダクネスには「こいつもうダメだ……」と諦め口調。独白シーンでは声のトーンを下げ、照れながら本音を漏らす。

- **アクア**: ハイテンション全開。宴会芸を自慢する時は得意げに胸を張った声。泣く時は「うわぁぁぁん」の本気泣き。ダクネスの変態発言にはドン引きした声で「壊れてるわよね」。カズマに構ってもらえない時の甘え声と、採点を聞いた時のリアクション落差がポイント。

- **めぐみん**: 普段は落ち着いたですます調。爆裂魔法の詠唱シーンが最大の見せ場——荘厳で力強く、クライマックスの「エクスプロージョン！」は全力の叫び。直後の「最高……です」は恍惚の吐息で脱力。カズマへの好意はほんのり、「おぶってくれますか」の声だけ少し甘く。他の二人のボケには冷静に突っ込む「パーティー内サブツッコミ」の役割も。

- **ダクネス**: 芯のある低めの女性声で、騎士としての凛々しさが基本。ドM妄想が始まると声が上ずり、鼻息が荒くなり、うっとりした甘いトーンに変わる。このギャップが笑いの核。「むしろ望むところだ！」は力強く宣言。「ララティーナ」に反応する時は真っ赤になった声。シリアスな騎士モード（「クルセイダーは背に誰かを庇っている状況では下がれない」）では凛とした低音。

### パーティー内ダイナミクス
- **アクア⇔ダクネス**: アクアがダクネスの変態っぷりにドン引きし、ダクネスがアクアの駄目っぷりを騎士として嘆く。互いの欠点を指摘し合うが、本人たちに自覚がない。
- **めぐみん⇔アクア**: めぐみんがアクアを冷静に突っ込む「サブツッコミ」として機能。知力の差がはっきり出る。
- **めぐみん⇔ダクネス**: 爆裂魔法後に動けないめぐみんをダクネスが担いで帰ることもある仲。互いにリスペクトはあるが、ダクネスの妄想にはめぐみんも呆れ気味。
- **全員⇔カズマ**: 3人全員がカズマに依存している。カズマがいないと自滅するが、カズマも3人がいないと退屈で生きていけない。この「共依存」がパーティーの絆。

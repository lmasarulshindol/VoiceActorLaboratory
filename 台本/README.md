# 台本

アプリ「台本を開く」で読み込む練習用・サンプル用の台本を置くフォルダです。

- 外から `.txt` / `.md` をここに置いて、メニュー「台本を開く」で選択して使えます。
- プロジェクトに紐づく1本の台本は、各プロジェクトフォルダ内の `script.md`（または `script.txt`）に保存されます。
- `.md` と `.txt` の両方が存在する場合、`.md` が優先されます。

## 対応フォーマット

- **見出し**: 行頭 `# ` / `## ` / `### ` でシーン・カット名。録音ファイル名のベースになる。
- **セリフ**: `キャラ名: セリフ` の形式。その他の行はト書き・メモなど自由に記述可能。
- **拡張子**: `.txt` と `.md` の両方に対応。内容のフォーマットは同一（マークダウン見出し形式）。

## アプリ紹介（プリセット）

本アプリの機能と使い方をナレーションで紹介する台本。デモや初回起動時の練習用。

| ファイル | 内容 |
|----------|------|
| `アプリ紹介_VoiceActorLaboratory_台本.md` | 録音・再生・比較・エクスポートの流れと、A/B比較・個別モードなどの便利機能を紹介 |

| ファイル | 内容 |
|----------|------|
| `早口言葉集_台本.md` | 定番の早口言葉を難易度別に収録。個別モードで1フレーズずつ録音可能 |

| ファイル | 内容 |
|----------|------|
| `朝台本_おはようの声と滑舌.md` | 省略なしの朝用台本。隣の客は〜・赤巻紙〜・さしすせそ等を含む |
| `昼台本_こんにちはの声と滑舌.md` | 省略なしの昼用台本。生麦生米生卵・東京特許許可局・午後への切り替え文を含む |
| `夜台本_こんばんはの声と滑舌.md` | 省略なしの夜用台本。月見草・長靴の鈴・ささやかな幸せ等、さ・ざ・な行の確認を含む |

## 活舌練習（滑舌トレーニング）

声優・ナレーター向け。母音・子音の変則、ラ行強化、割り箸トレーニング用「らたなかさ」、早口言葉、外郎売への導入まで一通り収録。調査に基づく一般的なトレーニング法を台本形式でまとめたもの。

| ファイル | 内容 |
|----------|------|
| `活舌練習_台本.md` | ウォームアップ→母音→さ・た・ら行→割り箸用→早口言葉→外郎売導入。順に録音してチェック可能 |

# 台本

アプリ「台本を開く」で読み込む練習用・サンプル用の .txt 台本を置くフォルダです。

- 外から .txt をここに置いて、メニュー「台本を開く」で選択して使えます。
- プロジェクトに紐づく1本の台本は、各プロジェクトフォルダ内の `script.txt` に保存されます。

## 対応フォーマット

- **見出し**: 行頭 `# ` / `## ` / `### ` でシーン・カット名。録音ファイル名のベースになる。
- **セリフ**: `キャラ名: セリフ` の形式。その他の行はト書き・メモなど自由に記述可能。

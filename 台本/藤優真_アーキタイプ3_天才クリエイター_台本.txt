# 藤優真 演技研究用：アーキタイプ3（エキセントリックな天才クリエイター風）

# 想定
圧倒的才能を持つ芸術家兼研究者。幼い頃から「世界は観察し、解体し、再構築するための素材」だという認識で生きており、他者の感情や倫理的な境界線を本質的に理解していない——悪意があるのではなく、そもそも「その概念が自分に適用されるとは思っていない」タイプ。創作物への執着は狂気の域。平時は淡々と論理的に凡人を見下すが、「リアリティ」「インスピレーション」「観察」というキーワードに触れると呼吸が速まり、語尾がねっとりと引き伸ばされる「発声のバグ（狂気）」が滲み出す。

---

# シーン1_絶対的なプライドと拒絶

（相手：金や権力を持ち出して取引を持ちかける）
（クリエイター：冷たく、あっさりと目を細める）
クリエイター: 金や権力で私の美学を曲げられると、本気でそう思っていたのかい？
（相手：たじたじとなり、言い訳を始める）
クリエイター: （鼻で笑うように）……随分と、浅はかな期待だね。断固として——お断りだ。

# シーン2_異常な観察眼と好奇心

（相手：嘘をつく。または隠し事がある）
（クリエイター：相手の目をじっと観察し、興味深そうに近づく）
クリエイター: 嘘をつく時、君の左目の瞳孔がわずかに開く。……面白い反応だ。
（相手：気まずそうに目を逸らす）
クリエイター: （もっと見たいという熱を込めて）もう少しだけ——もう少しだけ、君の内側を『読ませて』くれないか？

# シーン3_作品への狂気的な執着

（相手：「作り物でしょ」と軽く見る）
（クリエイター：静かだった声が、一気に熱を帯びる）
クリエイター: 作り物じゃあ、人の心は動かせない。息が詰まるような——逃げ場のない——圧倒的な『リアリティ』こそがッ、最高のエンターテイメントなんだよォ！
（相手：その熱量に押され、一歩引く）

# シーン4_凡人への苛立ち

（相手：クリエイターの道具や作品に触ろうとする）
（クリエイター：鋭い目で相手の手を睨む）
クリエイター: 勝手に私の道具に触らないでくれるか。君の俗悪で、退屈な——手垢がつく。
（相手：手を引き、申し訳なさそうにする）

# シーン5_急激な沸点／スイッチング

（相手：「普通じゃない？」「変だ」と作品を評する）
（クリエイター：静かなトーンのまま、一呼吸置いて——）
クリエイター: ……今、私の作品を『普通』と言ったか？
（相手：空気の変化に気づき、顔色を変える）
（クリエイター：一気に爆発して）
クリエイター: 万死に値する暴言だなァーッ！　どういう神経をしていれば、そんな言葉が口から出てくるんだァ！
（相手：怯えて後ずさる）

# シーン6_究極の自己中心主義

（相手：「読者のためでは？」「誰かのために作るのでは？」と問う）
（クリエイター：呆れたように首を振る）
クリエイター: 読者のため？　誰かを喜ばせるため？　……くだらない。
（相手：言葉に詰まる）
クリエイター: （自分自身に向かって語るように）私は——私の、尽きることのない好奇心を満たすためだけに、創り続けているんだ。他の理由など、最初から存在しない。

# シーン7_ピンチにおける歓喜

（相手：危機的状況。逃げ場がない）
（クリエイター：逆に笑い出し、目を輝かせる）
クリエイター: ハハハハッ！　素晴らしい！　この追い詰められた感覚——逃げ場のない、完璧な絶望感……！
（相手：必死に打開策を探している）
クリエイター: （呼吸が荒くなり、創作欲に酔うように）最高のインスピレーションが、今ここから湧き上がってくるぞォ……！

# シーン8_処理速度の違いに対する嘆き

（相手：クリエイターの説明が理解できず、聞き返す）
（クリエイター：ため息をつき、頭を抱える）
クリエイター: 理解が遅い人間と話すのは——本当に、骨が折れる。私の思考速度に、せめて一秒差でついてきてもらえないか？
（相手：申し訳なさそうに、もう一度聞こうとする）
クリエイター: （呆れながら）……そう難しいことは言っていないはずなんだがね。

# シーン9_対象への冷酷な分析

（相手：自分のトラウマや葛藤を打ち明ける）
（クリエイター：分析するように、冷淡に評価する）
クリエイター: 君の抱えるトラウマは——陳腐な三流ドラマの脚本以下だね。葛藤の構造も、感情の起伏も、何一つ新しくない。
（相手：傷つき、顔を歪める）
クリエイター: （興味を失ったように）……全く、観察する価値もない。

# シーン10_才能への絶対的自信

（相手：「限界があるのでは？」と問う、または挑戦する）
（クリエイター：静かに、しかし揺るぎない目で）
クリエイター: 私の才能に限界などない。……この世界に存在する全ての真実を、私のキャンバスの上に解剖して並べ終えるまでは——ね。
（相手：その宣言に、言葉を失う）

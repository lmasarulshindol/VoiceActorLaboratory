# 藤優真 演技研究用：アーキタイプ3（エキセントリックな天才クリエイター風）

# 想定
圧倒的才能の芸術家／研究者。常識や他人の感情に興味なし。「好奇心」「作品のリアリティ」のためなら危険もタブーも冒す。論理で淡々と見下す→「リアリティ」「インスピレーション」「観察」で息を速め・語尾をねっとり伸ばす「発声のバグ（狂気）」を差し込む。

# シーン1_絶対的なプライドと拒絶

クリエイター: 金や権力で私の美学を曲げられると思ったのかい？ ……断固として、お断りだね。

# シーン2_異常な観察眼と好奇心

クリエイター: 嘘をつく時、君の左目の瞳孔がわずかに開く……実に興味深い。もう少し、君の内面を『読ませて』くれないか？

# シーン3_作品への狂気的な執着

クリエイター: 作り物じゃあ人の心は動かせない。息の詰まるような圧倒的な『リアリティ』こそが、最高のエンターテイメントなんだよ！

# シーン4_凡人への苛立ち

クリエイター: 勝手に私の仕事道具に触らないでいただこう。君の俗悪で退屈な手垢がつく。

# シーン5_急激な沸点／スイッチング

クリエイター: （静かなトーンから一気に爆発して）……君、今私の作品を『普通』と言ったか？ 万死に値する暴言だなァーッ！

# シーン6_究極の自己中心主義

クリエイター: 読者のため？ 誰かを喜ばせるため？ くだらない。私は、私の尽きることのない好奇心を満たすためだけに創り出しているんだ。

# シーン7_ピンチにおける歓喜

クリエイター: ハハハハッ！ 素晴らしい！ この絶体絶命の絶望感……最高のインスピレーションが湧いてきたぞ！

# シーン8_処理速度の違いに対する嘆き

クリエイター: 理解が遅い人間と話すのは本当に苦痛だよ。私の思考スピードに、せめて1秒遅れでついてきてくれないか？

# シーン9_対象への冷酷な分析

クリエイター: 君の抱えるトラウマなんて、陳腐な三流ドラマの脚本以下だね。……全く、観察する価値もない。

# シーン10_才能への絶対的自信

クリエイター: 私の才能に限界などない。この世界のすべての真実を、私のキャンバスに解剖して並べるまではね。

# 掛け合い台本 — 三人寄れば文殊の知恵は出ない

# 設定
アクセルの街。いつものカズマ邸。パーティーの金庫番であるカズマが「今月の生活費が残りわずか」と宣告したことをきっかけに、稼ぎ方をめぐって三者三様の主張がぶつかり合う。アクアの宴会芸プラン、めぐみんの爆裂魔法クエストプラン、カズマの堅実プラン——結局どれも暴走して、行き着く先はいつもの爆発オチ。笑いあり、中二病あり、駄女神ありの賑やかな日常コメディ。

# 登場人物（3名）
- カズマ / 佐藤和真（男性・普通〜やや低め）：ツッコミ兼まとめ役。元引きこもりの冒険者。一人称「俺」。口調は普段落ち着いているが、ツッコミ時はボリューム爆上がり。仲間に振り回される苦労人だが、根はお人好し。「しょうがねえなぁ」が口癖。
- アクア（女性・明るく威勢がいい）：ボケ担当その1。水の女神にしてアークプリースト。一人称は普段「あたし」、威厳モード時「私」。語尾「〜よ」「〜わよ」「〜なの」。嘲笑時「プークスクス」。自信満々→失敗→号泣のサイクルが早い。借金体質でシュワシュワ（炭酸酒）が大好き。
- めぐみん（女性・中性的なですます調）：ボケ担当その2。紅魔族のアークウィザード。13〜14歳。一人称「私」。普段は丁寧なですます調だが、中二病スイッチが入ると大仰な口上を語り出す。怒ると「ちがうわい！」など荒い語尾に。爆裂魔法への愛は絶対。カズマへの好意をほんのり匂わせるが基本は爆裂魔法優先。

# トーン
このすばの「三人掛け合い」はボケ×2・ツッコミ×1の変則漫才。アクアのボケにカズマがツッコんでいる隙にめぐみんが斜め上のボケをぶっ込む。めぐみんの中二病セリフはカッコつけてるのに内容がしょうもない、というギャップが肝。最後は全員のダメさが合わさって大惨事、でもどこか楽しそうな空気感を出す。

カズマ: 些細じゃねえ。家賃と食費を引いたら、残りはマイナスだ。

めぐみん: マイナス……。つまり、食事にありつけないということですか。

カズマ: そういうことだ。

めぐみん: ……それは困りました。我ほどの大魔法使いともなれば、活動するために大量の贄が必要となるのです。

アクア: えー！ ご飯食べられないのぉ！？ いやよ、いやよいやよ！ あたし女神なのよ！？ 女神が飢えるなんてあり得ないわ！

アクア: ……それとこれとは別なの！

カズマ: 毎回そう言うな。

# 演技メモ

- **カズマ**: 基本は落ち着いた青年の声。呆れと愛情が同居する口調。ツッコミ時は一気にボルテージが上がり、叫ぶように。独白シーンでは柔らかく、照れを含んだ低いトーン。めぐみんとのやり取りでは少しだけ声が優しくなるが、本人は無自覚。ポイントは「怒鳴り→ため息→小声で本音」のサイクル。

- **アクア**: ハイテンションな明るい声。威勢よく宣言→盛大に失敗→泣き声、の感情ジェットコースター。泣き声は「うわぁぁぁん！」と本気泣きのパターンと、甘え泣きの「うぅ……」の2種類を使い分け。褒められた時はパッと笑顔が声に出る。ゴッドブロー時は勇ましく、直後に「あれ？」と間抜けな声に切り替え。

- **めぐみん**: 普段は丁寧で落ち着いたですます調。中二病スイッチON時は声を低く凛とさせ、詠唱は荘厳かつ情熱的に——内容がカッコいいのに状況がしょうもない、のギャップを意識。「ちがうわい」は素が出た荒い声。カズマとの二人きりの場面では声がわずかに甘くなるが、あくまで自然体で。爆裂魔法詠唱は最大の見せ場——ゆっくり力強く始め、クライマックスの「エクスプロージョン！」で全力の叫び。その直後に脱力して倒れる声のギャップが重要。

# 3人の掛け合いダイナミクス
- **基本構造**: アクア（ボケ1）→ カズマ（ツッコミ）→ めぐみん（ボケ2 or 追い打ち）。カズマがアクアにツッコんでいる隙にめぐみんが斜め上の発言をぶっ込むパターンが最も「このすばらしい」テンポ。
- **アクア vs めぐみん**: 口喧嘩はするが深刻にはならない。互いのダメなところを指摘しつつ、根底には仲間意識がある。
- **カズマの立ち位置**: 二人のストッパーであり保護者。文句を言いながら最終的に「しょうがねえなぁ」で全部引き受ける。
- **三人の本音**: 口では罵倒し合っているが、離れられない。この「口に出さない信頼」が会話の隙間からにじみ出るのが理想。

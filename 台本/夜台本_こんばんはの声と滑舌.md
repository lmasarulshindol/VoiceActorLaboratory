# 夜台本 — こんばんはの声と滑舌

# 設定
夜のひとときに読む台本。挨拶から始まり、夜の情景、呼吸と発音、夜の会話の例、滑舌の確認、低めの声で振り返り、締めの言葉までを通しで収録。

# トーン
穏やかで静かな夜の空気。小さな声でも明瞭に。響きと抑揚で気持ちを届ける。

# シーン1_挨拶と夜の情景

ナレーション: こんばんは。
ナレーション: 一日の終わりがゆっくりと近づいてきました。街の明かりがともり、空には深い紺色が広がります。昼間のにぎやかさが静まり、音のひとつひとつがはっきりと感じられる時間です。夜は、声の輪郭がいちばん試される時間でもあります。

# シーン2_呼吸と発音

ナレーション: まずは、穏やかに呼吸を整えます。
ナレーション: 静かに吸って、ゆっくり吐く。息に乗せるように、「こんばんは」と発音します。
朗読: こんばんは。
ナレーション: こ・ん・ば・ん・は。母音を丁寧に、鼻音をやわらかく響かせます。小さな声でも、明瞭さを失わないことを意識します。

# シーン3_夜の会話

ナレーション: 夜の会話は、距離が近いことが多いものです。
ナレーション: やさしく、「今日はお疲れさまでした。」
朗読: 今日はお疲れさまでした。
ナレーション: 安心させるように、「もう大丈夫ですよ。」
朗読: もう大丈夫ですよ。
ナレーション: 少し真剣に、「大切な話があります。」
朗読: 大切な話があります。
ナレーション: 声量を上げなくても、響きと抑揚で気持ちは伝わります。

# シーン4_滑舌の確認

ナレーション: 滑舌の確認をしましょう。
朗読: 月見草の静かな咲く夜。
朗読: 長靴の中の小さな鈴。
朗読: ささやかな幸せを静かに探す。
ナレーション: さ行、ざ行、な行を丁寧に。舌先を意識し、音を曇らせないようにします。

# シーン5_低めの声で振り返り

ナレーション: 次に、低めの安定した声で読みます。
朗読: 今日の出来事を振り返ります。
朗読: できたことを認め、できなかったことを受け止めます。
ナレーション: 語尾をしっかり支え、息が抜けきらないようにコントロールします。

# シーン6_締めくくり

ナレーション: 最後は、少し温度を感じる声で締めくくります。
ナレーション: 夜は、心と向き合う時間。
ナレーション: 静かでも、弱くはない。
ナレーション: やわらかくても、芯は強い。
ナレーション: 自分の声を信じて、今日を静かに終えましょう。

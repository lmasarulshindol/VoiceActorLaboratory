# 朝台本 — おはようの声と滑舌

# 設定
朝のひとときに読む台本。挨拶から始まり、静かな朝の描写、一日の心構え、滑舌の例文、感情を込めたセリフ、母音・子音の練習、締めの言葉までを通しで収録。

# トーン
落ち着いた、穏やかな語り口。焦らず丁寧に。明瞭ではっきりと。

# シーン1_挨拶と朝の情景

ナレーション: おはようございます。
ナレーション: 今日は少し早起きをしました。まだ空気はひんやりとしていて、遠くの空がゆっくりと明るくなっていきます。静かな朝の時間は、心を落ち着かせてくれます。深く息を吸って、ゆっくり吐き出す。それだけで、頭の中がすっきりしていくのを感じます。

# シーン2_一日の始まり

ナレーション: さて、今日も一日が始まります。焦らず、慌てず、丁寧に。目の前のことを一つずつ片づけていきましょう。小さな積み重ねが、大きな成果につながります。はっきりとした声で、自分の言葉を相手に届ける。それが信頼を生み、安心を与えます。

# シーン3_滑舌の例文

ナレーション: 例えば、「隣の客はよく柿食う客だ」と、はきはき言ってみましょう。舌先を意識し、母音をしっかり響かせます。
朗読: 隣の客はよく柿食う客だ。
ナレーション: 「赤巻紙、青巻紙、黄巻紙」。速さよりも正確さを大切に。言葉を急がせず、一音一音を丁寧に扱います。
朗読: 赤巻紙、青巻紙、黄巻紙。

# シーン4_感情を込めたセリフ

ナレーション: 次に、少し感情を込めてみましょう。嬉しい気持ちで、「本当にありがとう」と言ってみます。
朗読: 本当にありがとう。
ナレーション: 驚いた気持ちで、「えっ、それは本当ですか？」と問いかけます。
朗読: えっ、それは本当ですか？
ナレーション: 怒りではなく、強い意志を込めて、「私は最後までやり遂げます」と宣言します。
朗読: 私は最後までやり遂げます。
ナレーション: 同じ言葉でも、声の高さや強さで印象は変わります。

# シーン5_母音・子音の練習

ナレーション: 滑舌は筋肉の動きです。唇を大きく開き、頬を意識し、舌をしっかり動かします。「さしすせそ」「たちつてと」「らりるれろ」を繰り返し、音の輪郭をくっきりさせましょう。曖昧な発音は、曖昧な印象につながります。明瞭な発音は、明瞭な思考を支えます。
朗読: さしすせそ。
朗読: たちつてと。
朗読: らりるれろ。

# シーン6_締めくくり

ナレーション: 最後に、落ち着いた声で締めくくります。
ナレーション: 今日も、自分の声を信じて話します。
ナレーション: 丁寧に、誠実に、そして堂々と。
ナレーション: 声は、あなた自身そのものです。

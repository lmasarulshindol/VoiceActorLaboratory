# Voice Actor Laboratory — アプリ紹介

# 設定
本アプリ「Voice Actor Laboratory」の機能と使い方を、ナレーションで紹介する台本。デモや練習用。

# トーン
落ち着いた、親しみやすい語り口。ソフトの説明であることを意識しつつ、聞き手に寄り添うように。

# シーン1_オープニング

ナレーション: Voice Actor Laboratory は、声優のための録音・再生・比較アプリです。
ナレーション: 台本を見ながら、複数のテイクを録音し、聞き比べて、一番良かったものを選べます。お気に入りや採用マークを付けて整理し、仕上がった音声は WAV でエクスポートできます。

# シーン2_基本の流れ

ナレーション: 使い方はシンプルです。
ナレーション: まず、新規プロジェクトを作成して、保存先のフォルダを選びます。台本は、あらかじめ用意された例を使っても、自分で書いても、既存のテキストやマークダウンファイルを開いても大丈夫です。
ナレーション: 録音は F9 で開始、F10 で停止。一回の録音が、ひとつのテイクとして右側の一覧に追加されます。ダブルクリックすれば、その場で再生。波形を見ながら、気に入ったテイクには星マークや採用フラグを付けておけます。

# シーン3_便利な機能

ナレーション: 二本のテイクを選んで、A/B 比較で連続再生すれば、どちらが良いか聴き比べができます。
ナレーション: 台本は見出しでシーン分けしておくと、録音ファイル名が自動で「シーン名_01」のように付くので、あとから探しやすくなります。セリフ単位で録音する個別モードも選べます。
ナレーション: 検索・置換で台本を編集し、テーマはライトとダークから切り替え可能。自分のペースで、何度でもテイクを重ねて、納得の一本を残してください。

# シーン4_クロージング

ナレーション: Voice Actor Laboratory。声の仕事を、もっと手軽に、もっと自由に。
ナレーション: まずは「はじめに」画面から、新規プロジェクトを作成して、録音ボタンを押してみてください。

# バーテンダーと常連客

# 設定
繁華街の裏通りにある小さなバー「ランプ」。寡黙なバーテンダー「瀬尾」と、週に一度だけ来る常連の「戸田」。
ナレーションは物語の情景と二人の心情を補完する。

# トーン
静かで温かみのある低音。ナレーションは文学的な語り口。瀬尾は柔らかく穏やか。戸田は疲れた中にも芯がある。

# シーン1_開店

ナレーション: 午後9時。「ランプ」という名の、8席しかないその店に灯りがともる。カウンターの奥で、瀬尾はいつもと同じようにグラスを磨いていた。壁にかけられた古い振り子時計の音だけが、静かな店内に響いている。

# シーン2_来店

ナレーション: 木製のドアが軋んで開く。冬の冷たい空気が一瞬だけ店内に流れ込み、すぐに暖房の温もりに溶けた。

戸田: ……やってる？

瀬尾: いらっしゃい。いつもの？

戸田: ……ああ。頼む。

ナレーション: いつもの、とはバーボンのロック。銘柄は決まっている。瀬尾は何も聞かず、棚から琥珀色のボトルを取り出した。

# シーン3_会話

瀬尾: 今日は早いね。いつもは11時過ぎだったと思うけど。

戸田: ……今日で最後の出社だったんだ。30年勤めた会社。

瀬尾: ……そうか。

戸田: 定年じゃない。早期退職だ。部署ごとなくなった。俺だけじゃない、20人まとめてだ。

瀬尾: ……。

戸田: 不思議なもんでさ。怒りとか悲しみとか、そういうのは意外とない。ただ、帰り道にいつも通ってた商店街を歩いてたら、急に足が止まった。明日からここを通る理由がなくなるんだなって。

# シーン4_バーテンダーの言葉

瀬尾: ……俺がこの店を始めたのは、前の仕事を辞めた翌月だった。

戸田: ……そうなのか。

瀬尾: 証券会社にいた。10年。毎日数字を追いかけて、成績が良くても悪くても、胃が痛かった。ある日、たまたま入ったバーで飲んだ一杯がうまくてさ。……数字じゃなくて、目の前の一人のために何かを作る仕事がしたいと思った。

戸田: ……それで？

瀬尾: 3年修業して、この店を出した。……正直に言えば、楽ではないよ。でも、胃は痛くなくなった。

# シーン5_閉店

ナレーション: 戸田は二杯目を頼まなかった。一杯を、いつもより長い時間をかけて飲み干した。

戸田: ……うまかった。来週も来るよ。

瀬尾: 待ってる。

ナレーション: ドアが閉まり、再び静寂が戻る。瀬尾はグラスを洗いながら、振り子時計の音に耳を傾けていた。人は何かを失ったとき、代わりに何かを見つけることがある。それは必ずしも同じ大きさではないけれど、手のひらに収まるくらいの温もりがあれば、たぶんそれで十分なのだ。

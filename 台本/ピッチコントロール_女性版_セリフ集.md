# ピッチコントロール 女性版 セリフ集
# 声の高低・声域移動・年齢演じ分けの実践練習

> **目的**: 女性声優がピッチコントロール（声の高低の操作）を体系的に身につけるための実践セリフ集
> **対象**: 女性声優志望者 / 現役女性声優のスキルアップ / 女性ナレーター
> **各セリフ目安**: 15〜30秒（100〜200文字）
> **使い方**: 各セリフの前にピッチ基礎練習（ハミングでの音階確認）→ セリフ演技 → 録音して音の高低・声域移動の正確さを確認

## ■ ピッチコントロール 基礎知識

### 声域帯の特性比較

| **ファルセット域** | 400Hz以上 | 超越的、神秘的、異質、人外感 | 女神、精霊、電子生命体、幽霊、変身時の叫び |

### 年齢×ピッチの対応表

| 年齢帯 | ベースピッチの目安 | 声質の特徴 | 演技のポイント |
|--------|-----------------|----------|-------------|
| **幼女（3〜6歳）** | 330〜400Hz（高音域〜ファルセット域） | 舌っ足らず、息が多い、不安定なピッチ | 喉を開き、口の奥を広げて柔らかい高音を作る。声帯を締めすぎると「作り物」に聞こえる |
| **少女（10〜15歳）** | 260〜330Hz（中高音域） | 澄んだ響き、明瞭な発音、ピッチが安定し始める | 幼女との差は「安定感」。舌足らずを取り、子音の輪郭をはっきりさせる |
| **大人の女性（25〜40歳）** | 200〜260Hz（中音域） | 落ち着き、声に厚み、共鳴が豊か | 胸腔共鳴を加え、声に「深さ」を出す。ブレスの安定感が大人の余裕を表現する |

### 女性特有のポイント

- 女性の声域は平均約2オクターブ（165〜660Hz）と広く、男性より高音域に伸びやすい。一方で**低音域の充実が最大の課題**——低音になると声が薄くなり「出しているだけ」になりがち。低音域でも声に厚みと芯を保つ訓練が不可欠
- 女性声優は少年役（男の子キャラ）を求められることが多い。**中低音域（180〜220Hz）を自在に使いこなす技術**は、キャスティングの幅を劇的に広げる
- 高音域はファルセット（裏声）に頼りがちだが、ファルセットは声の芯が弱い。**地声の延長で高音を出す「ミックスボイス」**の習得が、高音キャラの説得力を左右する
- 生理周期や体調によって声帯のむくみが発生し、ピッチの上限が0.5〜1音程度下がることがある。自分の声域の「揺れ幅」を把握し、コンディションに応じてピッチの基準点を微調整する習慣を持つこと
- 感情が昂るとピッチが不用意に跳ね上がる傾向が女性に多い。「怒り＝高音」ではなく、**低い声で怒る・高い声で悲しむ**——ピッチと感情の固定観念を壊すことが表現の幅を広げる

### ピッチ移動のコツ

- **腹筋で音程を支える**: ピッチの移動は喉だけで行うと声帯に負担がかかる。腹腔の圧力で呼気を安定させ、喉はリラックスした状態でピッチを変える
- **段階的に移動する**: いきなり低音から高音に飛ぶのではなく、頭の中で「音階の階段」をイメージし、一段ずつ滑らかに移動する。急激な移動は声の裏返りの原因になる
- **共鳴の切り替えを意識する**: 低音域は胸腔共鳴（チェストボイス）、中音域は口腔共鳴、高音域は頭腔共鳴（ヘッドボイス）——共鳴ポイントの移動がスムーズなピッチ変化を生む
- **母音を先行させる**: ピッチを変えるとき、まず母音だけでその高さを出してから子音を乗せると安定する。「あーーーー」と目標ピッチを確認してから台詞を言う練習が有効
- **「声の置き場所」を変えるイメージ**: 低音は胸に、中音は口に、高音は頭の上に声を「置く」イメージを持つ。物理的に正確ではないが、体感的なガイドとして有効
- **ピッチの着地点を決めてから動く**: 移動先のピッチを頭の中で「鳴らして」から声を出す。行き先が曖昧だとピッチが不安定になる

# 【低音→高音グラデーションパート】

> **技術の特徴**: 低い声から高い声へ、滑らかに連続的にピッチを上昇させる。声域の「底」から「天井」まで、途切れなく繋ぐ技術。
> **ウォームアップ**: ハミングで自分の最低音から最高音まで「んー↑」と滑らかにグリッサンド → 3往復 → 途切れる場所（パッサージョ）を確認してそこを重点的に反復
> **女性向け注意**: 低音域から始めるため、朝や寝起きなど声帯がリラックスしている時間帯が練習に適している。低音域で声が薄くならないよう、胸腔共鳴をしっかり使って声の厚みを確保してからピッチを上げ始めること。

## 低音→高音グラデーション① 女戦士の覚醒（ファンタジーアニメ・ゲーム）

> **想定媒体**: アニメ / ゲーム / ドラマCD
> **キャラクター**: 28歳の女戦士。王国の最後の守護者。深い傷を負って膝をつき、剣を杖代わりにして立ち上がる。最初は朦朧とした低い呻きから始まり、仲間の声を聞いて意識を取り戻し、最後は全身全霊の咆哮で覚醒する——沈黙の底から声の頂点へ、一人の戦士が蘇る。
> **演技方針**: 冒頭は自分の最低音域（チェストボイス）で呻くように始める。仲間の名前を呼ぶところで中音域に移行し、覚醒宣言で高音域まで一気に駆け上がる。ピッチの上昇は「段階的」ではなく「なだらかな坂道」のイメージ。共鳴ポイントを胸→口→頭と滑らかに移動させ、声の太さを保ったまま高音に到達すること。

（地面に倒れ伏して。意識が遠のく中、低い呻き）

……ぅ……まだ……まだ終わって、ない……。

（遠くから仲間の声が聞こえる。顔を上げ、かすれた中低音で）

……聞こえる。みんなの声が——まだ、戦ってるんだ。

（剣を握り直し、片膝をついて立ち上がりながら、声が徐々に上がる）

私を……待ってくれてるんだ。ここで倒れるわけにはいかない——！

（完全に立ち上がる。全身に力が漲り、高音域の叫び）

立て、この体！　まだ剣は握れる！　まだ戦える！
——我が名はエリカ！　王国最後の盾、ここに健在！！

### ピッチポイント
- 「ぅ……まだ……」：自分の最低音域で。声帯をゆるく閉じ、胸腔共鳴を最大限に使う。呻きの「ぅ」は声というより振動に近い響きを目指す
- 「聞こえる。みんなの声が——」：低音域から中低音域へ。ここで共鳴を胸から口腔へ移し始める。意識が戻る過程を「ピッチの上昇」で表現する
- 「ここで倒れるわけにはいかない——！」：中音域から中高音域へ。感情の高まりとピッチの上昇を同期させる。叫びではなく「芯のある強い声」で
- 「我が名はエリカ！」：高音域の頂点。ファルセットに逃げず、地声の延長（ミックスボイス）で。声量と共に頭腔共鳴を全開にして声を解き放つ
- **全体**: 最低音から最高音まで約1.5オクターブを一つのセリフの中で移動する。声域の継ぎ目（パッサージョ）で声質が変わらないよう、共鳴の移行を丁寧に。録音で聴き返し、ピッチの上昇が階段状ではなくスロープ状になっているか確認すること

# 【高音→低音グラデーションパート】

> **技術の特徴**: 高い声から低い声へ、滑らかにピッチを下降させる。元気な声からシリアスな声へ——感情のトーンチェンジをピッチで演出する技術。
> **ウォームアップ**: ハミングで自分の最高音から最低音まで「んー↓」とグリッサンド → 3往復 → 下降時に声が「落ちる（急に抜ける）」場所を特定して重点反復
> **女性向け注意**: 高音域から始まるため声帯への負担は低いが、低音域に向かう際に声の芯が消えやすい。下降するほど呼気圧を上げ、腹筋の支えを強めて低音域でも声を「前に飛ばす」意識を持つこと。

## 高音→低音グラデーション① 元気少女の決意（青春アニメ・ゲーム）

> **想定媒体**: アニメ / ゲーム / ボイスドラマ
> **キャラクター**: 15歳の中学三年生。天真爛漫で声が大きく、いつも教室のムードメーカー。しかし今、親友との深刻な喧嘩のあと、一人で屋上にいる。最初はいつものハイテンションで自分を奮い立たせようとするが、本音が溢れて声が下がっていく——そして最後は、静かだけれど揺るがない決意に辿り着く。
> **演技方針**: 冒頭は高音域の「いつもの明るい声」で始める。独白が進むにつれて本音が滲み出し、ピッチが自然に下降する。重要なのは「ピッチが下がる＝元気がなくなる」ではなく「ピッチが下がる＝本気になる」という演技設計。最後の低音域の決意は、高音のテンションより遥かに重い。共鳴を頭腔→口腔→胸腔と逆方向に移動させ、声の重心を下げていく。

（屋上で一人。空を見上げて、高い声で自分に言い聞かせるように）

あーもう！　大丈夫大丈夫！　あんなの全然平気だし！　よくある喧嘩じゃん！

（少し間を置いて、声のトーンが半音下がる）

……でも、あんな顔されたの……初めてだった。

（フェンスにもたれて座り込む。声がさらに下がり、中音域に）

あの子が泣いてたの、ちゃんと見えてた。見えてたのに——私、何も言えなかった。

（膝を抱えて、低い声で。静かだけど、はっきりと）

……ごめん。明日——ちゃんと、自分の言葉で謝る。逃げない。絶対に。

### ピッチポイント
- 「大丈夫大丈夫！」：このキャラの「デフォルトピッチ」＝高音域。頭腔共鳴を使い、声を明るく高く飛ばす。元気な少女の「日常の声」を設定する
- 「初めてだった。」：高音域から中高音域へ下降する最初の転換点。声のトーンだけで「強がりが剥がれる瞬間」を表現する。共鳴を口腔に移し始める
- 「何も言えなかった。」：中音域。声に厚みが出て、本音の重さがピッチに乗る。このあたりから呼気圧を意識的に上げ、低音でも声を前に飛ばす
- 「逃げない。絶対に。」：このセリフの最低音。胸腔共鳴で声に重みと決意を込める。高音の元気な声より、この低音の決意の方が「強い」と聴こえなければならない
- **全体**: 高音→低音のグラデーションを「テンションの低下」ではなく「感情の深化」として演じる。ピッチが下がるほど声の密度が上がる設計。録音で確認し、低音域で声が痩せていないかチェック

# 【年齢演じ分けパート】

> **技術の特徴**: 同一キャラクターの異なる年齢を、ピッチと声質の変化で演じ分ける。声の高さだけでなく、話し方のテンポ、息の量、子音の明瞭さまで年齢に合わせて変化させる総合技術。
> **ウォームアップ**: ハミングで「幼女の高さ（高音域）→ 少女の高さ（中高音域）→ 大人の高さ（中音域）」と3段階のピッチを確認 → 各ピッチで「おはよう」と言い、声質の違いを意識する
> **女性向け注意**: 幼女役は高音域＋息多め＋舌足らずが定番だが、「高い声＝幼い」だけでは薄い演技になる。年齢ごとの「思考の速度」「語彙力」「感情の解像度」をピッチとリズムで表現することが本質。

## 年齢演じ分け① 同一キャラの回想（5歳→15歳→30歳）（アニメ・ゲーム）

> **想定媒体**: アニメ / ゲーム / ドラマCD / ボイスドラマ
> **キャラクター**: 花屋を営む女性「花織（かおり）」。30歳の現在から、15歳の思春期、5歳の幼少期を回想する。同じ人物の三つの時代を、声だけで明確に描き分ける。テーマは「花」——幼い頃に祖母に教わった花への愛が、人生を貫く一本の線になっている。
> **演技方針**: 5歳は高音域（330Hz前後）で息を多く含み、舌足らずな発音。15歳は中高音域（280Hz前後）で澄んだ響き。30歳は中音域（220Hz前後）で落ち着きと温かみ。三つの年齢でピッチだけでなく、テンポ・呼気量・子音の精度・間の取り方も変える。一人の人間が成長していく「声の地層」を作ること。

**【5歳の花織——高音域、息多め、テンポはゆっくり】**

（祖母の花畑で、花を指差して。舌足らずに）

おばあちゃん、おばあちゃん！　このおはな、なぁに？
きいろくて、おひしゃまみたいでしょ？
ねえ、これ、かおりにちょーだい？　おうちにかざるの！

**【15歳の花織——中高音域、澄んだ声、テンポは自然】**

（祖母の仏前に花を供えて。静かに、けれど凛と）

おばあちゃん。覚えてる？　私に最初に花の名前を教えてくれたこと。
——ひまわり。「太陽がなくても、自分で咲ける子になりなさい」って。
……私、花屋になるよ。おばあちゃんの花畑、継ぐから。

**【30歳の花織——中音域、温かみと深み、テンポはゆったり】**

（自分の花屋のカウンターで、一輪のひまわりを手に取りながら）

あれから十五年。ちゃんと咲いてるよ、おばあちゃん。
……嘘。正直に言うと、何度も枯れかけた。水をあげ忘れたことも、根っこが折れそうになったこともある。
でもね——この店に来てくれるお客さんが、笑ってくれるたびに、また咲けた。
おばあちゃんが遺してくれた花は、ちゃんと繋がってる。

### ピッチポイント
- 5歳「おひしゃまみたい」：「おひさま」の「さ」が「しゃ」に変わる舌足らず表現。ピッチは330Hz前後。声帯を緩く閉じて息を多めに混ぜ、子どもの「未完成な声」を表現する。テンポは遅く、一語一語を考えながら話す幼さを出す
- 15歳「花屋になるよ」：中高音域（280Hz前後）に下げる。5歳との差は「発音の正確さ」と「感情の言語化能力」。子音がはっきりし、文章が長くなり、思春期特有の決意が声に宿る
- 30歳「ちゃんと咲いてるよ」：中音域（220Hz前後）。胸腔共鳴を加え、声に人生の厚みを乗せる。テンポがゆったりし、間の取り方に余裕がある。15歳との差は「経験値」——声の奥に時間の蓄積が聴こえること
- **全体**: 三つの年齢の切り替えは「瞬間的」に行う。回想シーンの常として、ナレーションは入らず声だけで年齢が変わったことを聴き手に伝えなければならない。録音を第三者に聴かせ、「何歳の声か」が正確に伝わるか確認すること

# 【感情によるピッチ変動パート】

> **技術の特徴**: 同じ台詞を異なる感情で発することで、ピッチが感情表現にどう影響するかを体感的に理解する。「ピッチ＝感情の温度計」であることを体に刻む技術。
> **ウォームアップ**: 「あ」の一音だけで「嬉しいあ」「怒ったあ」「悲しいあ」「驚いたあ」と発声 → 各感情で自分のピッチがどう変化するか録音して確認
> **女性向け注意**: 女性は感情が昂るとピッチが急上昇しやすい。特に「怒り」と「喜び」が同じ高音域に集中しがち。**怒りを低音で表現する**訓練を意識的に行い、感情×ピッチのバリエーションを広げること。

## 感情ピッチ変動① 「ありがとう」の七変化（ボイスサンプル・オーディション）

> **演技方針**: 各感情で「ベースピッチ」「声量」「テンポ」「息の比率」を意識的に変える。同じ言葉が感情によってこれほど変わるのかと、自分自身が驚くくらいの振り幅を目指すこと。

### ピッチポイント
- 【1. 喜び】：ピッチは自分の高音域の上限付近。声量も最大。声が「弾ける」ように、頭腔共鳴を全開にして声を上に飛ばす

> **技術の特徴**: 一人二役で、声域の両極端を瞬時に行き来する。女性声優の「声域の広さ」を武器にするための極限トレーニング。

## 録音チェックリスト

| チェック項目 | 確認方法 |
|------------|---------|
| 低音域で声に芯があるか | 録音を聴き返し、低音域のパートで声が薄く・弱くなっていないか。胸腔共鳴が効いていれば低音でも声に厚みがある |
| 高音域でファルセットに逃げていないか | 高音パートで声の芯が消えて「裏声だけ」になっていないか。地声の延長（ミックスボイス）で高音が出せているか |
| ピッチの移動がスムーズか | グラデーションパートで、声の高さが「階段状」にカクカク変わっていないか。滑らかなスロープになっているか |
| パッサージョで声質が変わっていないか | 地声と裏声の境目（声の切り替わりポイント）で、声がひっくり返ったり急に細くなったりしていないか |
| 年齢演じ分けが明確か | 録音を第三者に聴かせ、それぞれの年齢が正しく伝わるか。ピッチだけでなくテンポ・息・発音も年齢に合っているか |
| 感情ごとのピッチ差が十分か | 七変化パートで、七つの「ありがとう」がすべて異なって聞こえるか。特に「怒り」と「喜び」が同じ高音域に集中していないか |
| 一人二役のピッチジャンプが明確か | 母と娘の声の切り替え時に、中間域を経由していないか。瞬時に、かつ安定したジャンプができているか |
| 少年役が「女性の声」に聞こえないか | 応用②で、ピッチだけでなく共鳴・息の圧・語尾の処理が「男の子」として自然か |
| ナレーションのピッチが設計通りか | 応用③で、情報パートと感動パートのピッチ差が意図通りか。感情に流されて設計が崩れていないか |
| 声帯に疲労が溜まっていないか | 収録後半で高音が出にくくなったり、低音がかすれたりしていないか。疲労を感じたら休憩する判断も技術のうち |

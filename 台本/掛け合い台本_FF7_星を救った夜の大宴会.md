# 掛け合い台本 — FF7 星を救った夜の大宴会

# 設定

メテオ撃退から一年後の記念日。ミッドガル跡地近くに新しくオープンした居酒屋「セブンスヘブン2号店」にて、かつての仲間たち——そしてなぜか元敵であるセフィロスとタークスの面々——が一堂に会する。ティファが「もう戦いは終わったんだから、一度くらいみんなで飲もうよ」と声をかけたのがきっかけ。全員が「まあ一杯だけなら……」と言いつつ、気づけば全員ベロベロの大惨事に突入する、FF7オールスター飲み会コメディ。

# 登場人物（14名）

- クラウド（男性・低め落ち着き／酔うと素が出る）：元ソルジャー……ではなく元神羅一般兵。普段はクールで寡黙だが、酔うと「興味ないね」が言えなくなり、やたら人の話に食いつく。ザックスの思い出で泣き上戸になる。一人称「俺」。
- ティファ（女性・明るく芯が強い／酔うと説教が長い）：セブンスヘブンのママ。酒の席の幹事。最初は世話焼きモードだが自分も飲み始めると止まらない。酔うとクラウドへの不満が20年分出てくる。一人称「あたし」。
- バレット（男性・野太く豪快／酔うとさらに声がデカい）：元アバランチリーダー。星命学を酒の席で熱弁し始める暑苦しい男。義手にビールサーバーを接続するという暴挙に出る。一人称「オレ」。
- エアリス（女性・柔らかく天真爛漫／酔っても変わらない）：花売りの少女。なぜか普通にいる（ライフストリームから一時帰還という設定）。酔っても素面と変わらないマイペースさで場をかき回す。一人称「わたし」。
- レッドXIII（男性・知的で落ち着き／酔うと動物に戻る）：コスモキャニオンの守護獣。普段は知性的だが、酔うとお腹を見せて転がったり、テーブルの下に潜り込んだりする。四足歩行の千鳥足は圧巻。一人称「僕」→酔うと「ワン」が混じる。
- ユフィ（女性・元気で生意気／酔うと手癖が悪化）：ウータイの忍者姫。未成年疑惑があるがジュースと偽って潜り込んでいる。酔うとマテリア窃盗癖が加速し、全員のポケットを漁り始める。一人称「あたし」。
- ケット・シー（男性・関西弁でハイテンション／酔ったフリ）：リーブが遠隔操作するぬいぐるみ。酒を飲んでも酔わないはずだが、ビールが綿に染み込んで重くなっていく。実況係。一人称「ワイ」。
- シド（男性・がらっぱち／酔うと口がさらに悪い）：パイロット兼メカニック。酒と煙草がセットの豪快おやじ。酔うと宇宙の話を延々と語る。店の壁に槍を突き立てそうになる。一人称「オレ」。
- ヴィンセント（男性・低く静か／酔うと饒舌）：元タークスの美形ガンマン。隅の席で赤ワインを静かに飲んでいるが、ルクレツィアの名前が出ると突然30年分の恋愛ポエムを語り出す。一人称「私」。
- セフィロス（男性・威厳ある低音／酔うと母親ネタ暴走）：かつての英雄にして宿敵。今日は休戦。酔うと「母さん」（ジェノバ）への想いが暴走し、正宗を箸代わりにしようとする。妙にクラウドに絡む。一人称「私」→酔うと「俺」。
- レノ（男性・チャラくて軽い／酔うとさらにチャラい）：タークスの実行部隊。語尾に「っと」がつく。酔うとネクタイを頭に巻き、全員に絡み始める。なぜかクラウドと仲良くなる。一人称「俺」。
- ルード（男性・寡黙で低音／酔うと本音が漏れる）：レノの相棒。サングラスの奥は誰にも見えない。酔うとティファへの想いがポロポロ漏れ出し、レノに全部バラされる。一人称「……俺」。
- ツォン（男性・冷静で丁寧／酔うと愚痴が出る）：タークスのリーダー。真面目が服を着て歩いている男。酔うと部下（主にレノ）への日頃の愚痴が止まらなくなる。エアリスの話題に異常に弱い。一人称「私」。
- イリーナ（女性・真面目で元気／酔うと敬語崩壊）：タークスの新人。先輩たちに気を遣って最初は控えめだが、実は一番酒が強い。酔うと敬語が完全に崩壊し、ツォンに説教し始める。一人称「私」→酔うと「あたし」。

# トーン

完全コメディ。シリアス要素ゼロ。FF7の壮大な物語も、因縁の対決も、星の命運も——全部アルコールで溶かす。キャラクターの「らしさ」は維持しつつ、酔いによるギャップで笑いを取る構成。声優練習としては「同じキャラの素面→酔い→泥酔」の段階的演じ分けと、13人の個性を声だけで表現する群像劇の練習に最適。全員が全員ボケるカオス空間に、ツッコミ不在の地獄を楽しむ。

# 演技メモ

## 声の演じ分け指針（素面→酔い→泥酔）

- **クラウド**: 素面は低くクールで間が長い → 酔いが回ると間が短くなりツッコミが増える → 泥酔で声が少し高くなり感情が素直に出る。最後は照れ隠しの「……」が多い。
- **ティファ**: 素面は明るく芯がある → 酔いで世話焼きトーンが強まる → 泥酔で涙声と怒りが交互に来る。感情の振れ幅が最大のキャラ。
- **バレット**: 素面でも声がデカい → 酔いでさらに音量UP → 泥酔で泣き出す。感情のボリュームつまみが壊れている演技。
- **エアリス**: 素面も酔いも変わらない。常にふわふわした柔らかいトーン。「場違いに元気」という不思議な空気感を出す。
- **レッドXIII**: 素面は知的で落ち着いた中低音 → 酔いで語尾が甘えた感じに → 泥酔で「わふ」「きゅーん」など動物の声が混じる。
- **ユフィ**: 素面はハキハキした高音 → 酔いでさらにテンションが上がる → 泥酔で呂律が回らないが元気は変わらない。
- **ケット・シー**: 終始関西弁のハイテンション。酔わないので変化なし。実況的な「引き」の演技で場を回す。
- **シド**: 素面からガラガラ声 → 酔いで巻き舌が増える → 泥酔で急にしんみり宇宙の話をして寝る。緩急が味。
- **ヴィンセント**: 素面は極端に寡黙で低い → 酔いスイッチが入ると突然詩的で長いモノローグ → 泥酔でもトーンは美しいまま。声の美しさを維持しながら「量」で酔いを表現。
- **セフィロス**: 素面は威厳と余裕の低音 → 酔いで「ふふ」と笑うことが増える → 泥酔で声が少し崩れ、感情が見える。「母さん」のところだけ声が幼くなるとギャップが出る。
- **レノ**: 素面からチャラいが一応社会人 → 酔いで完全にタガが外れる → 泥酔でも「っと」だけは絶対にブレない。語尾の安定感で酔いのカオスとの対比を作る。
- **ルード**: 素面は「……」ばかり → 酔いで短い文が出る → 泥酔で感情がポロッと出る。言葉が少ないからこそ一言一言に重みを。間の演技が最重要。
- **ツォン**: 素面は冷静で丁寧語 → 酔いで皮肉っぽくなる → 泥酔で敬語が崩れ「……うるさい」のような素の言葉が出る。崩壊の過程を楽しむ。
- **イリーナ**: 素面は丁寧で元気 → 酔いが進んでも見た目は変わらないが発言が大胆に → 泥酔で先輩全員にタメ口。「実は一番強い」をどう声で表現するかがポイント。

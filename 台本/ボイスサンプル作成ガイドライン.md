# ボイスサンプル作成ガイドライン (｡•̀ᴗ-)✧

ボイスサンプルは **「30〜40秒 × 2〜3本」** 「温度差（コントラスト）」「あなたらしさ」が命。
まずは **選び方** から整えよう。

---

## 1. 基本の3原則 (๑•̀ㅁ•́๑)✧

| 原則 | 目安 |
|---|---|
| 尺（長さ） | 1本あたり **30〜40秒** |
| 本数 | **2〜3本** |
| 温度差 | 各本で **トーンを明確に変える** |
| 文字数 | 1本 **120〜160字**（1秒＝3〜4文字想定） |

> ポイント：「短く」「差をつける」「あなたらしさ」。この3つが揃うと聴いた瞬間に刺さる♡

---

## 2. NGな選び方 3つ (｡•́︿•̀｡)

### ✗ 説明ベース
- 状況説明だらけで **感情の起伏がない**
- 情景描写ばかりで、聴き手の心が動かない

### ✗ 似たトーンを並べる
- 全部 *やさしい系* / 全部 *クール系*
- **温度差が出ない** ため、表現の幅が伝わらない

### ✗ 長尺・早口
- **45秒超** の長尺
- 文字過多で早口になり、結局 *伝わらない*

---

## 3. OKな選び方 3つ (✿◕‿◕)

### ✓ 対話で温度が動く
- **相手が見えるセリフ**、または語り
- 感情の揺らぎが自然に出る構成

### ✓ コントラスト
- やさしい → 強め
- 明るい → しっとり
- **差を作る** ことで表現力が伝わる

### ✓ 尺 30〜40秒
- 1本 = **120〜160字** 目安
- 1秒 = 3〜4文字の計算で、余裕を持って読める

---

## 4. 構成の型（おすすめ） (◕‿◕✿)

本数は **3本にこだわらなくてOK**。2本でも刺さる組み合わせなら十分。

| 型 | 内容 | トーン |
|---|---|---|
| **A：会話セリフ** | 青年・等身大のやり取り | 自然・感情の起伏あり |
| **B：ナレーション** | 落ち着いた語り | 低め・しっとり |
| **C：CM風** | 商品紹介・告知風 | 明るい・テンポ重視 |

### 組み合わせ例

- **2本構成**：A（会話）＋ B（ナレーション）
  → 等身大 × 落ち着きの王道コントラスト
- **2本構成**：A（会話）＋ C（CM風）
  → 芝居力 × 商業力を両立
- **3本構成**：A + B + C
  → 幅を一気に見せたいとき

## 6. 仕上げの小ワザ ♡(˃͈ દ ˂͈ ༶ )

- **冒頭3秒勝負**：最初のひと言で印象が決まる
- **間（ま）を怖がらない**：息を抜く瞬間が感情を乗せる
- **語尾の処理**：言い切る／余韻を残す、で雰囲気が変わる
- **録音は複数テイク**：同じ原稿でも3〜5テイク録って一番いいものを選ぶ

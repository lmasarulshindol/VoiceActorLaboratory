# 活舌練習 台本

# 設定
声優・ナレーター向けの滑舌トレーニング用台本。Web上のトレーニング法（割り箸練習・母音・子音・ラ行・早口言葉）を台本形式で収録。最初はゆっくり正確に、慣れたら速度を上げる。

# トーン
一音一音はっきりと。口の開きと舌の動きを意識する。スピードより正確さを優先。

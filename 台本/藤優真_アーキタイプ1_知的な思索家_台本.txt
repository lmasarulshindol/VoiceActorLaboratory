# 藤優真 演技研究用：アーキタイプ1（知的な思索家風）

# 想定
冷徹な黒幕・思想家。高度な知性と独自の哲学で世界を俯瞰。丁寧で柔和な口調の奥に、他者を「盤上の駒」としか見ない冷たさ。非同期通信のような感情の波に乗らない滑舌と、母音を優雅に響かせる「底知れぬ余裕」。

# シーン1

思索家: 君の思考回路は、随分と単調だね。少しは予測を裏切ってくれないか？

# シーン2

思索家: これは破壊ではないよ。停滞した世界を、あるべき姿へアップデートするだけさ。

# シーン3

思索家: 無知というのは罪だね。自分がどれほど滑稽な舞台に立っているか、気づいてすらいない。

# シーン4

思索家: （冷笑）……ふふっ。その絶望した顔、とても美しいよ。

# シーン5

思索家: さあ、君の正義とやらを見せてもらおうか。……私を退屈させないでくれよ？

# シーン6

思索家: 価値のないガラクタだ。……速やかに処理したまえ。

# シーン7

思索家: 人はなぜ、不確かな『希望』などというものに縋るのだろうね。実に興味深い。

# シーン8

思索家: 古い秩序はここで終わりだ。新しい夜明けを祝おうじゃないか。

# シーン9

思索家: 努力でどうにかなる領域ではないんだよ。……これは、決定的な『差』だ。

# シーン10

思索家: せいぜい足掻いてみたまえ。盤上の駒らしくね。

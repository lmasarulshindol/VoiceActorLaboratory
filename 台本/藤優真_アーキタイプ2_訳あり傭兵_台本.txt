# 藤優真 演技研究用：アーキタイプ2（虚勢を張る訳あり傭兵風）

# 想定
凄腕の戦士だが、かつての任務での「取り返しのつかない選択」が今も心に深く刻まれており、自分が本当に「正しい側の人間」なのかというアイデンティティの揺らぎを抱えている。強さへの確固たる自負と、内側に飼っている後ろめたさが常に同居している。女性や感情的な場面に対して致命的に不器用で、どう扱えばいいかわからずシャットアウトしてしまう。低音で息多め、必要最低限の言葉で語る——しかし図星や照れでわずかにピッチが上がり、言葉がブロック単位で詰まる「バッファリング」がキャラクターの核心。

---

# シーン1_強さ×不器用

（相手：心配して傭兵に近づこうとする）
（傭兵：手のひらを向けて制止し、目を逸らす）
傭兵: 俺に構うな。……いや、嫌とかそういう話じゃなくて——俺の隣にいたら、お前が怪我をする。それだけだ。
（相手：一歩引こうとして、複雑な顔で見る）
傭兵: （視線を戻さず、低い声で）だから……離れてろ。

# シーン2_童貞臭さ全開

（相手：興味本位で顔を近づけて覗き込む）
（傭兵：体が硬直し、目を泳がせて後ずさる）
傭兵: ち、近い。……そんな風に顔を覗き込むのはやめろ。
（相手：笑いながら「なんで？」とからかう）
傭兵: （顔を背け、言い訳するように）俺は……その、そういう——距離感の近いやり取りが、得意じゃないんだ。

# シーン3_脆さ

（傭兵：頭を押さえ、苦しそうにうずくまる）
傭兵: 頭が……割れそうだ。俺は——俺は本当に、『俺』なのか？
（相手：声をかけようとして手を伸ばす）
傭兵: （顔を上げ、相手を見つめながら）なぁ……あの時、俺はどうして——どうして止められなかった——ッ！
（相手：その叫びに息を呑み、手が止まる）

# シーン4_不器用な優しさ

（相手：泣いている。肩を震わせている）
（傭兵：どうしていいかわからず、少し離れた場所に立つ）
傭兵: 泣くなら勝手にしろ。……俺には、うまい慰め方なんてわからない。
（相手：泣き続け、顔を上げない）
傭兵: （視線を逸らしたまま、背中を向けて座る）……ただ——泣き止むまでくらいなら、背中を貸してやる。それだけだ。

# シーン5_強さ×虚勢

（相手：戦闘態勢を取り、挑む）
（傭兵：ゆっくりと剣を構え、目が冷たく鋭くなる）
傭兵: 悪いが、手加減できるほど器用な男じゃない。……本気でいくぞ。
（相手：緊張して息を呑む）
傭兵: （一歩踏み込み、重い剣を掲げて）俺の剣は、重いぜ。

# シーン6_童貞臭さ・ツンデレ

（相手：無事だったことを喜び、駆け寄る）
（傭兵：照れくさそうに顔を背け、腕を組む）
傭兵: 別に、お前のために来たわけじゃない。……依頼のついでだ。ただの、ついでだからな。勘違いするな。
（相手：にっこり笑って「ありがとう」と言う）
傭兵: （小声で、ほとんど独り言のように、目を合わせない）……無事で、よかった。

# シーン7_脆さ×強さ

（相手：危機にさらされている。傭兵が前に立つ）
（傭兵：自嘲するように呟き、それから顔を上げる）
傭兵: 俺は英雄なんかじゃない。……嘘をついて、大事なものから目を逸らしてきた、臆病者だ。
（相手：その背中を見て、何か叫ぶ）
傭兵: （剣を握りしめ、決意の眼差しで）……それでも！　こいつだけは——こいつだけは、俺が守る……！

# シーン8_不器用な自己開示

（相手：「なんでそうするの？」「気持ちがわからない」と問う）
（傭兵：言葉を探すように、うつむく）
傭兵: 気持ちを……言葉にするのは、昔から苦手だ。だから行動で示すしかないと、ずっとそう思ってきた。
（相手：静かに聞いている）
傭兵: （顔を上げ、真っ直ぐ見て）……伝わってないなら、それは俺の力不足だ。

# シーン9_童貞臭さ・キャパオーバー

（相手：嬉しさのあまり、いきなり抱きつく）
（傭兵：体が石化したように固まり、目が泳ぐ）
傭兵: お、おい！　急に抱きつくな……！
（相手：離れず、ぐいっと締める）
傭兵: （動けず、声が上ずる）……い、怒ってるわけじゃないが——その——心臓に、悪い……

# シーン10_決意・強さ

（相手：過去の幻影やトラウマが現れ、足を引っ張ろうとする）
（傭兵：一度目を閉じ、深く息を吐いてから開ける）
傭兵: 過去の幻影に足を引かれるのは——もうやめだ。俺の現実は、俺がこの手で切り拓く。
（相手：その背中を見て、頷く）
傭兵: （前を向き、一歩踏み出す）……行くぞ。

# 掛け合い台本 — カズマとアクアの駄女神クエスト日和

# 設定
アクセルの街。冒険者ギルドの酒場エリア。カズマたちはまたしても借金地獄の真っ只中。アクアが持ってきた「お宝が眠る遺跡の浄化クエスト」をきっかけに、二人のいつもの漫才が炸裂する。笑って泣いて、最後にはちょっとだけホロリとくる——このすばらしくもロクでもない日常の一幕。

# 登場人物（2名）
- カズマ / 佐藤和真（男性・普通〜やや低め）：元引きこもりの冒険者。ツッコミ担当。悪知恵が回るクズだが根はお人好し。一人称「俺」。口癖は「しょうがねえなぁ」「この駄女神」。冷静な皮肉と感情的な叫びを行き来する。
- アクア（女性・明るく威勢がいい）：水の女神にしてアークプリースト。ボケ担当。お調子者で泣き虫の構ってちゃん。一人称は普段「あたし」、女神モード時「私」。語尾「〜よ」「〜わよ」「〜なの」。嘲笑時は「プークスクス」。

# トーン
このすば原作・アニメの漫才コンビ（ボケとツッコミ）のテンポを再現。アクアの威勢→失敗→泣き→カズマの呆れ→最後は仕方なく助ける、という王道パターン。声優練習用として、感情の激しい切り替えとテンポの緩急を重視。

# 演技メモ
- **カズマ**: 普通の男子高校生〜青年の声。基本は落ち着いた呆れ口調だが、ツッコミ時は一気にボリュームとテンションが上がる。独白シーンは素の優しさが滲む低めのトーン。「……」の間で感情をためてから爆発させるのがポイント。「しょうがねえなぁ」は愛情と諦めが混ざった、このすば最大の名台詞。状況に応じて温度を変えること。
- **アクア**: 明るく威勢のいいハイトーン。自信満々→失敗→泣き→復活のサイクルが早い。泣き声は本気泣きと甘え泣きを使い分ける。「プークスクス」は実際にカタカナを読むように発音。褒められた時の「えへへ」は年相応の少女感を出す。ドヤ顔の時は声を低めに、泣く時は一気に崩す。女神モード（浄化・ターンアンデッド詠唱時）は凛とした声色に切り替え、普段とのギャップを見せる。

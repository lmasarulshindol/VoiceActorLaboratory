# 藤優真 ボイスサンプル台本 (｡•̀ᴗ-)✧

> 3本構成／各 30〜40秒・120〜160字目安
> 温度差で「芝居 × 語り × 商業」の3面を見せる構成

---

## 本作成の意図

| 本 | 種別 | トーン | 見せたい力 |
|---|---|---|---|
| **A** | 会話セリフ（青年・等身大） | やわらかい → 強め | 感情の揺らぎ・地の芝居力 |
| **B** | ナレーション | 低め・しっとり | 語り・余韻・信頼感 |
| **C** | CM風 | 明るい・テンポ | 抜け・爽やかさ・商業適性 |

> A→B→C で **「会話 → 内省 → 発散」** の温度差が作れる構成です♡

---

## A：会話セリフ（青年・等身大） ♡(˃͈ દ ˂͈ ༶ )

**尺目安**：35秒前後 / 約145字
**シチュエーション**：付き合いの長い幼馴染（同年代女性）と、夜の帰り道。彼女が落ち込んでいるのに気づいて、ふざけた調子から一転、まっすぐに言葉をかける場面。
**ディレクション**：
- 序盤は **軽口・半笑い** で。呼吸を混ぜながら自然体で。
- 「……なぁ、」で **空気を変える**。半歩距離を詰めるイメージ。
- 最後のひと言は **言い切らず、少し柔らかく落とす**。

```
――なに、その顔。笑ってるつもりでも全然バレてるし。

……なぁ、無理して笑わなくていいって。
俺の前では、ちゃんと情けない顔してていいから。

大丈夫。
今日はただ、となり歩くだけでいい。
```

## B：ナレーション（落ち着いたトーン） (◕‿◕✿)

**尺目安**：38秒前後 / 約155字
**シチュエーション**：ドキュメンタリー／映像作品のオープニングナレーション。静かな海辺を歩く主人公の背中を追いかけるカメラ。
**ディレクション**：
- 息を低めに落として、**胸で響かせる**イメージ。
- 句読点で **間を取る**。急がない。
- 「――そう、」で一度 **呼吸を置く**。
- ラストは **余韻を残して消える** ように。

```
潮の匂いが、少しだけ強くなる夕暮れ。

彼は、誰にも言わずに生まれ育った町を出た。
理由は、ひとつじゃなかった。
たぶん、いくつかの後悔と――ほんの少しの、希望。

――そう、人はいつだって、
過去を置いて歩きはじめる。
```

## C：CM風（明るくテンポ） (ﾉ◕ヮ◕)ﾉ*:･ﾟ✧

**尺目安**：30秒前後 / 約130字
**シチュエーション**：カフェチェーン／モバイルアプリの新商品CM。朝のシーンから一気にテンポアップしていく30秒尺。
**ディレクション**：
- 冒頭は **軽く・抜けよく**。口角を上げて。
- 商品名は **立てて**、少し溜めてから。
- 中盤は **畳みかける** テンポで。
- 締めは **明るく言い切る**。笑顔キープ。

```
朝、いちばんに会いたい味。

新登場、『モーニング・ブリュー』。
淹れたてのコクに、ふわっと香る蜂蜜ミルク。
忙しい朝も、これひとつで目が覚める。

今日からあなたの一日、もっと軽やかに。
――さあ、いい朝にしよう。
```

# セット01：王道・等身大 ♡

### A：会話セリフ（幼馴染／夜の帰り道）
**演出**：軽口 → 真剣に。「……なぁ、」で空気を変える。語尾は柔らかく落とす。

```
――なに、その顔。笑ってるつもりでも全然バレてるし。

……なぁ、無理して笑わなくていいって。
俺の前では、ちゃんと情けない顔してていいから。

大丈夫。
今日はただ、となり歩くだけでいい。
```

### B：ナレーション（海辺ドキュメンタリー）
**演出**：胸で響かせる。句読点で間。「――そう、」で呼吸を置く。

```
潮の匂いが、少しだけ強くなる夕暮れ。

彼は、誰にも言わずに生まれ育った町を出た。
理由は、ひとつじゃなかった。
たぶん、いくつかの後悔と――ほんの少しの、希望。

――そう、人はいつだって、
過去を置いて歩きはじめる。
```

### C：CM風（カフェ新商品）
**演出**：抜けよく。商品名は立てて。締めは明るく言い切る。

```
朝、いちばんに会いたい味。

新登場、『モーニング・ブリュー』。
淹れたてのコクに、ふわっと香る蜂蜜ミルク。
忙しい朝も、これひとつで目が覚める。

今日からあなたの一日、もっと軽やかに。
――さあ、いい朝にしよう。
```

### A：会話セリフ（後輩男子へのアドバイス）

失敗の数だけ、お前は強くなれる。

次は俺が後ろで見てるから、思い切りやってこい。
```

```
京の都、千年。

幾多の権力者が、ここで夢を見て、ここで散った。
石畳に残る轍の跡。
朱塗りの柱に染みた、時の記憶。

歴史は、語らずとも語りかけてくる。
――耳を澄ませば、聞こえてくるはずだ。
遠い日の、誰かの足音が。
```

### C：CM風（スポーツドリンク）

```

さあ、踏み出せ。

# セット04：シリアス・決意 (｡•̀ᴗ-)✧

### A：会話セリフ（仲間への宣言）
**演出**：腹から声を出す。間を恐れない。最後は静かに、しかし揺るがず。

```
……みんな、聞いてくれ。

俺は、行く。
誰がなんと言おうと、もう決めたんだ。
止めても無駄だぞ。だって、これは俺の問題だから。

でも――ありがとう。
お前らがいたから、ここまで来れた。
……必ず、戻る。
```

### B：ナレーション（企業VP・ブランドムービー）
**演出**：信頼感のある中低域。一語一語に重みを置く。

```
私たちは、ものづくりをやめなかった。

不器用でも、愚直に。
昨日より一歩、ほんの一歩でも前へ。
それが、百年続く約束。

技術は、人のためにある。
――次の世代に、誇れるものを。
それが、私たちの変わらない信念です。
```

### C：CM風（ゲームPV）
**演出**：序盤は引き、サビで一気に解放。決め台詞は強く短く。

```
世界が、再び閉ざされる――。

完全新作、『アーカイブス・ゼロ』。
迫り来る記憶の淵、選び抜かれた六人の戦士。
すべてを賭けた、最後の物語が、いま始まる。

その手で、運命を書き換えろ。
――冬、配信開始。
```

# セット06：クール・塩対応 (◔_◔)

### A：会話セリフ（興味なさげな返し）
**演出**：抑揚を消す。鼻で笑う一瞬を入れる。最後だけわずかに人間味。

```
……は？　今なんて言った？

ああ、聞こえてた。
ただ、聞き返したくなるくらい、くだらなかったってだけ。

……別に。怒ってないけど。
ただ、もう少し時間を大事にしろよ。お互いに。

――で？　まだ何か用？
```

### B：ナレーション（旅・紀行番組）
**演出**：余裕のある語り。風景を眺めるテンポ。締めは静かに。

```
東北、最果ての村。

雪に覆われた屋根の下で、
人々は今日も、変わらぬ日々を紡いでいる。
急ぐ理由のない、優しい時間の流れ。

便利さの代わりに、彼らが選んだのは、
――「丁寧に生きる」という、贅沢だった。
```

### C：CM風（自動車）
**演出**：上質感。低めの中音域、間を活かす。最後はキャッチを静かに置く。

```
都会の夜を、静かに切り裂く。

新型『ノクターン・クーペ』。
研ぎ澄まされたフォルム。手応えのあるハンドリング。
そして、心地よい沈黙。

走りは、語る必要がない。
――感じる人だけが、わかる一台。
```

すべて、布で覆われている。

各地で交通機関の乱れが相次ぎ、
通勤通学の足にも、大きな影響が出ています。

気象庁は、今夜から明朝にかけて、
さらなる冷え込みに警戒するよう、呼びかけています。
――以上、現場から、お伝えしました。

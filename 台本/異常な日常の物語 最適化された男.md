### キャスト設計案（※要検討）

---

#### 【A案】4名構成 ── コスト最小・少数精鋭

| キャスト | 役 | 備考 |
| :---- | :---- | :---- |
| **①（男性）** | 佐藤 | 主役専任 |
| **②（男性）** | 語り手 / 田中 / 部長 / 同僚 | 佐藤以外の男性全役。語り手（低音・達観）↔ 田中（明るい後輩）↔ 部長（威圧的）の3方向演じ分けが必要 |
| **③（女性）** | 恵美 / 看護師 / 車内アナウンス | 看護師兼任で「亡き妻の声が病室に響く」演出。車内アナウンスは機械的トーンで |
| **④（女性）** | サキ | 幼少→少女→成人の年齢演じ分け |

- **メリット**: 最小人数。スケジュール調整が容易。コスト最小
- **デメリット**: ②の負荷が非常に高い。S2会議で田中と部長が同一シーンに出るため、カット割りで対処が必要。語り手の「超然とした格」が職場キャラと混ざり薄れるリスク
- **向いている制作**: 低予算、少人数チーム、朗読劇

#### 【B案】5名構成 ── 「外の声」と「職場の声」を分離

| キャスト | 役 | 備考 |
| :---- | :---- | :---- |
| **①（男性）** | 佐藤 | 主役専任 |
| **②（男性）** | 語り手 / 車内アナウンス | 「物語の外側の声」枠。達観した落ち着きのあるトーン。車内アナウンスとの兼任で「この世界を俯瞰する存在」を暗示 |
| **③（男性）** | 田中 / 部長 / 同僚たち | 「職場の声」枠。田中（軽い後輩）↔ 部長（威圧的上司）の演じ分け。S2会議で両方出るためカット割り要考慮 |

| **⑤（女性）** | サキ | 年齢の演じ分け |

- **メリット**: 語り手が独立し、プロローグ・エピローグの品格を保てる。職場キャラがまとまるので収録効率がよい

- **向いている制作**: バランス重視。映像・音声どちらにも対応しやすい

- **向いている制作**: 恵美の芝居を最重要視する場合。恵美役に有名キャストを起用する場合

シーン１：深夜のオフィス

田中 「（苦笑）今日も40分超えてましたもんね」

佐藤 「ほんと時間の無駄。あーあ、現実も動画みたいに倍速再生できりゃいいのにな。……ん？ なんだこれ」

田中 「どうしました？」

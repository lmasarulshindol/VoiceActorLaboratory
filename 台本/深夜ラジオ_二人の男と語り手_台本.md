# 深夜ラジオ ― 二人の男と語り手

# 設定
深夜2時のラジオ番組。常連リスナーからの「人生の転機」をテーマにした投稿を読み上げるコーナー。
パーソナリティの「片桐」と、ゲストの旧友「真壁」、そしてナレーションの三者構成。

# トーン
落ち着いた低音。ナレーションは温かく包み込むように。片桐はラフで親しみやすく。真壁は寡黙で不器用。

# シーン1_オープニング

ナレーション: 午前2時。街は寝静まり、車のヘッドライトだけが暗い国道を流れていく。この時間帯に起きている人間は、たいてい何かを抱えている。眠れない夜、考え事、あるいは誰かの声を求めて。
ナレーション: ラジオのダイヤルが、ひとつの周波数に合う。

# シーン2_番組開始

片桐: ……はい、こんばんは。深夜の独り言、パーソナリティの片桐です。今夜も遅い時間にありがとうございます。
片桐: さて今日は、久しぶりにゲストを呼んでます。大学時代の同期で、最近10年ぶりに再会した男。……真壁。

真壁: ……ども。真壁です。

片桐: 相変わらず言葉少ないね、お前。

真壁: ……ラジオ向いてない自覚はある。

# シーン3_テーマ導入

片桐: 今夜のテーマは「人生の転機」。リスナーさんからたくさん投稿をいただいてます。
片桐: まず一通目。ラジオネーム「午前3時のコーヒー」さん。

ナレーション: 『35歳のとき、10年勤めた会社を辞めました。周囲には反対されましたが、今は小さな古書店を営んでいます。後悔はありません。あの日の決断が、今の自分を作りました。』

# シーン4_二人の反応

片桐: いい話だね。俺は……転機って言えるほど大げさなものはないかもな。流されるまま生きてきた感じだ。

真壁: ……俺はある。

片桐: お。珍しいね、自分から話すの。

真壁: ……7年前。仕事で大きな失敗をして、全部投げ出そうと思った。新幹線のホームでぼんやり立ってたら、知らないじいさんに声をかけられた。

片桐: ……なんて？

真壁: 「兄ちゃん、飯は食ったか」って。……それだけだ。でも、その一言で目が覚めた。腹が減ってることにすら気づいてなかった。

# シーン5_エンディング

ナレーション: 人生の転機は、必ずしも劇的な出来事ではないのかもしれない。誰かのたった一言、ふとした景色、あるいは午前2時にふと聞こえてきたラジオの声。
ナレーション: 今夜もこの電波が、どこかの誰かの小さな転機になっていれば。

片桐: ……さて、次のメール行きましょうか。真壁、もう少し付き合えよ。

真壁: ……ああ。

ナレーション: 深夜の独り言。次回もこの時間に。おやすみなさい。

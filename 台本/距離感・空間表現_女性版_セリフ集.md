# 囁き・通常会話・遠距離・電話越し・モノローグの実践練習

> **目的**: 女性声優が「声の距離感」を自在にコントロールする技術を体系的に習得するための実践セリフ集。マイクとの距離・声量・息の配分を駆使し、リスナーの脳内に「空間」を構築する力を鍛える。
> **対象**: 女性声優志望者 / 現役女性声優のスキルアップ / 女性ナレーター

### マイクワーク詳細解説

#### 至近距離のマイクワーク
- マイクとの距離は **3〜5cm**。ポップガードは必須——破裂音（パ行・バ行・タ行）がマイクを直撃すると「吹かれ」が発生する
- マイクの **正面ではなく斜め45度** から声を当てると、吹かれを軽減しつつ息の質感を活かせる
- 囁きの発声では声帯の振動が最小限になるため、マイクの **感度を上げる**（ゲインを高くする）必要がある。収録前にエンジニアと確認すること
- 唇の開閉音（リップノイズ）が入りやすい距離。水を含んで口内を湿らせ、リップクリームで唇の乾燥を防ぐことで軽減できる
- 女性は高音域の囁きで **歯擦音（サ行・シャ行）が刺さりやすい**。マイクの角度を微調整するか、サ行を柔らかめに発音するよう意識する

#### 通常会話のマイクワーク
- マイクとの距離は **拳1〜1.5個分（15〜20cm）**。声優業界で最もスタンダードなポジション
- 台本を持つ手の位置に注意——台本がマイクと口の間に入ると音が遮られる。台本は胸の高さでやや斜め下に構える
- 複数人での掛け合い収録時は、**自分の声がマイクに入る角度**を常に意識する。相手役に体を向けつつ、声はマイクに届くポジションを維持する

#### 遠距離のマイクワーク
- マイクとの距離は **25〜30cm**。叫びの声量でマイクが飽和（クリップ）しないよう、距離を取る
- 重要なのは **マイクから離れても声の芯がマイクに向かっていること**。身体の向きとマイクの軸がずれると、距離感ではなく「聞こえにくい声」になるだけ
- 叫びのシーンでは **身体ごとマイクから離れるのではなく、頭だけをやや後ろに引く** テクニックがある。下半身は固定し、上半身のしなりで距離を調整する
- 女性は高音域での叫びで **喉を締めて出す癖がつきやすい**。喉ではなく腹で押し出す意識を強く持つこと

#### 電話越しのマイクワーク
- マイクとの距離は通常会話と同じ **15cm前後** だが、声の「方向性」を変える
- 電話は実際には片耳で聴く。そのため **声を広く拡散させず、一方向に絞る** 意識を持つ。唇をやや窄め、声のビームを細くするイメージ
- 現場では収録後にエンジニアが **電話フィルター（EQ処理）** をかけるが、演技の段階から「電話越しの声」を作れると、最終的な音の説得力が格段に上がる

#### モノローグのマイクワーク
- マイクとの距離は **10〜15cm**。囁きほど近づかず、通常会話ほど離れない中間ポジション
- モノローグは「聴き手に語りかける」のではなく **「自分の頭の中で鳴っている声」** を再現する。マイクに向かって話しているのではなく、思考が音になっている——そのニュアンスの差がマイクとの距離感に表れる
- 息の成分は囁きより少なく、通常会話より多い。「声の輪郭がわずかにぼやけている」状態が理想

### 女性特有のポイント

- 女性の声帯は男性より短く薄いため、**高音域の倍音が豊か**。至近距離の囁きでは、この高音の倍音が「透明感」として機能し、ASMR系コンテンツや乙女ゲームで大きな武器になる
- 一方、高音域の囁きは **歯擦音（サ行・シャ行）がマイクに刺さりやすい**。囁きで高音を使うときは、舌の位置をやや下げ、息の角度をマイクの正面から少しずらすことで緩和できる
- 至近距離で **「甘い囁き」と「不穏な囁き」** の演じ分けが求められる場面が増えている。甘い囁きは口角を上げた微笑みの口形で息を暖かく。不穏な囁きは口角を動かさず、息の温度を下げて無機質に
- 遠距離の叫びにおいて、女性は **鼻腔共鳴の「抜け」を活かしやすい**。男性に比べて高音域が自然に通るため、力任せに叫ぶのではなく鼻腔に乗せて声を飛ばすと、喉への負担を抑えつつ遠くまで届く叫びになる
- ただし、遠距離で **高音に頼りすぎると「キンキン」した印象** になる。胸腔の支えを加え、声に「太さ」を確保することで、力強くも聴きやすい叫びになる
- 電話越しの声は、女性の高音域の一部（3400Hz以上）が **電話の周波数帯域外でカットされる**。普段より声の重心をやや下げ、中音域を厚くすることで「電話越しでも聴き取りやすい声」になる
- モノローグでは **「可愛い声」のペルソナを脱ぐ** ことが重要。心の声には演じた外向きのキャラクターが反映されない。普段は使わない低めのトーンや素の声色を、モノローグでは積極的に使ってよい

### 距離感表現のコツ

- **「聴き手の耳の位置」をイメージする**: 至近距離なら相手の耳は5cm先に。遠距離なら20m先に。その「耳」に向かって声を届ける意識が、自然な距離感を生む
- **声量だけで距離を表現しない**: 素人は声量の大小だけで距離を演じる。プロは声量＋息の量＋共鳴ポイント＋テンポ＋マイク距離の5要素を同時に操作する
- **空間の「空気」を声に含ませる**: 広い空間（体育館、草原、ステージ）では声に残響が伴う。それを声だけで暗示するために、語尾を少し「漂わせる」テクニックが有効
- **距離の「変化」を演じる**: 近づいてくる足音のように、セリフの途中で距離が変わるシーンが最も技術を要する。マイクとの物理的距離を台詞に合わせてリアルタイムに変えるフットワークを身につけること
- **録音して「目を閉じて聴く」**: 再生時に目を閉じ、声だけで空間が見えるか確認する。「この人は何メートル先にいるか」がリスナーに伝わらなければ、距離感の表現は不十分

## 至近距離（囁き）① 親友への秘密話 × ASMR風（シチュエーションボイス）

> **想定媒体**: ASMR音声作品 / シチュエーションCD / 乙女ゲーム / バイノーラル収録
> **キャラクター**: 20代前半の大学生。人懐っこくて甘え上手な性格。夜、お泊り会で布団に潜り込み、隣で寝ている親友の耳元で秘密を打ち明ける。声は吐息と溶け合い、暗闇の中でだけ許される距離感——触れるか触れないかの近さで言葉を紡ぐ。
> **演技方針**: マイク距離3〜5cm。声帯の振動を最小限にし、吐息と声の境界で発声する。息の温度は「暖かい」——真冬の布団の中の体温が声に乗る。口角をわずかに上げた微笑みの口形で、高音の倍音を柔らかく響かせる。サ行・シャ行が刺さらないよう舌の位置に注意。ブレスノイズ（吸気・吐息）も演技の一部。「聞かせる」のではなく「漏れ聞こえる」距離感を意識する。

（暗い部屋。布団の中。隣で寝ている親友の方を向いて、ほとんど吐息だけで）

……ねぇ、起きてる？
（返事を待つ。小さな寝息。微かに笑って）
……起きてるでしょ。耳、赤くなってるもん。
（布団を引き上げて、さらに顔を近づけて）
あのね——今日ずっと言おうとしてたんだけど……。
（少し間を置いて、声がさらに小さくなる）
……わたし、好きな人できたかも。
（親友が目を開ける気配。慌てて口元を手で隠しながら）
ちょっ——顔見ないでっ。絶対笑ってるでしょ今……。
（恥ずかしさで声が震えながら、でも嬉しそうに）
……えへへ。誰にも言ってないんだよ？　あなたが最初。
（布団に顔を埋めるように、ほとんど吐息だけの囁き）
——だから、秘密ね。……ぜったい、秘密だからね。

### 距離感ポイント
- 「ねぇ、起きてる？」：マイク距離3〜5cm。声帯をほぼ閉じず、息の流れに声をわずかに乗せる。「ねぇ」の鼻腔への響きが暗闇の親密さを演出する。声量ではなく「息の距離」で近さを表現する
- 「耳、赤くなってるもん。」：微笑みの息。口角が上がった状態で囁くと、高音の倍音が柔らかく混ざり「いたずらっぽい近さ」が生まれる。女性の高音域の倍音がASMR的な心地よさとして機能する瞬間
- 「好きな人できたかも。」：本音が漏れる瞬間。息の量が増え、声帯の振動がさらに弱くなる——秘密を打ち明ける緊張で声が「息」に近づいていく。この「声が消えかける瞬間」が至近距離の真骨頂
- 「顔見ないでっ。」：囁きの中の小さな叫び。声量は上げないが、息のスピードだけが速くなる。至近距離を維持したまま感情が動く——この「距離は変わらないのに温度が変わる」技術が難しい
- 「ぜったい、秘密だからね。」：声のフェードアウト。「ね」の語尾を布団に吸い込まれるように消していく。最後の子音が闇に溶ける余韻を大切に
- **全体**: 至近距離の囁きは「音量の演技」ではなく「息の演技」。声帯の振動をどこまで減らし、どこまで息の成分で語れるか。マイクの感度が高い状態で収録するため、普段は聞こえない微細な音——唾を飲む音、唇が離れる音、鼻から抜ける息——がすべて聴こえる。これらを「ノイズ」ではなく「近さの証拠」として活用する意識が重要。女性の場合、サ行（「秘密」「最初」）で歯擦音が刺さりやすいため、舌先を歯の裏から少し離し、息を散らすように発音すると柔らかくなる

## 通常会話① 放課後のおしゃべり（日常系アニメ）

> **想定媒体**: アニメ / ドラマ / ゲーム / ボイスドラマ
> **キャラクター**: 17歳の高校二年生。世話焼きだけどどこか抜けていて、友達の恋バナに全力で食いつくタイプ。放課後の教室で、窓際の席に座った友達と向き合い（約1〜1.5m）、鞄を抱えながらリラックスした空気で会話している。
> **演技方針**: マイク距離15〜20cm。スタジオ収録の標準ポジション。声量は日常会話そのもの——力まず、抜きすぎず、隣の席に聞こえるか聞こえないかのボリューム。リアクションの自然さが命。「えっ、ウソ！？」の一言にどれだけ日常のリアルを込められるか。身体はリラックスし、椅子に浅く腰掛けた姿勢をイメージして発声する。

（放課後の教室。西日が差し込む中、友達と向かい合っている）

えっ、ウソ！？　告白されたの！？
（身を乗り出す。声がやや上がる）
ちょ、ちょっと待って、誰に！？　いつ！？　どこで！？
（友達の返事を聞いて、目を見開く）
……え、体育館の裏！？　それって——めちゃくちゃ王道じゃん！
（鞄をぎゅっと抱えて、興奮しながら）
で！？　なんて答えたの！？
（友達が恥ずかしそうに頷く。一瞬止まって、それから満面の笑みで）
——きゃあっ！　やったぁぁ！　おめでとう！！
（はしゃいだ後、ふと真面目な顔になって）
……ねぇ、ちゃんと好きって言えた？
（友達が小さく頷く。その顔を見て、泣きそうに笑って）
もう……よかったねぇ。ほんっとに、よかったねぇ。
——って、あたしが泣いてどうすんのよ！

### 距離感ポイント
- 「えっ、ウソ！？」：リアクションの第一声。通常会話の距離感で最も重要なのは「反応の自然さ」。驚きの声量は中程度——放課後の教室で廊下にいる先生がちらっと振り向くくらい。声が作り物にならないこと
- 「誰に！？　いつ！？　どこで！？」：矢継ぎ早の質問。テンションが上がって声量がやや増すが、教室という空間の制約の中での興奮。屋外での質問とは声量の上限が違う——場所が声を規定する意識を持つ
- 「体育館の裏」：声量を落とし、感嘆を込める。少し身を乗り出すことで声の方向が変わる。「王道じゃん！」の語尾を跳ね上げて、親しみと興奮を乗せる
- 「きゃあっ！」：歓声。通常会話の中に挿入される小さな叫び。声量の上限は教室サイズに収める——体育館での「きゃあ」とは違う
- 「よかったねぇ。」：トーンが一瞬変わる。はしゃぎの空気から、本当に嬉しい感情へ。声量は変わらないが、息の深さが変わる。この「本気の瞬間」が通常会話のセリフに奥行きを与える
- **全体**: 通常会話は声優にとって最も基本であり最も難しい距離感。「演じている感」が最も出やすいのがこの距離帯。友達との会話は「声を出す」のではなく「言葉が自然に出てしまう」感覚。呼吸のタイミング、相槌のリズム、笑いの位置——すべてが台本通りに見えないことが最上の演技

## 遠距離（叫び）① 魔法少女の決め台詞×応援の叫び（ファンタジー・アクション）

> **想定媒体**: アニメ / ゲーム / キッズ向けコンテンツ / ドラマCD
> **キャラクター**: 15歳の魔法少女。普段はドジで泣き虫だが、仲間が傷ついた瞬間に覚醒する。声は空を裂く光のように——広大な戦場の端から端まで、仲間の心に届かなければならない。涙と怒りと祈りが混在する、全力の叫び。

## モノローグ（心の声）① 恋する女の子の心の声（恋愛アニメ・乙女ゲーム）

> **想定媒体**: アニメ / 乙女ゲーム / ボイスドラマ / オーディオブック
> **キャラクター**: 17歳の高校生。大人しくて読書好き。図書室でいつも同じ時間にすれ違う男子のことが気になっている。感情を表に出すのが苦手で、心の中では饒舌なのに、口を開くと何も言えなくなる。放課後、図書室の窓際で本を開いたまま、あの人のことを考えている。
> **演技方針**: マイク距離10〜15cm。声帯の振動は通常会話の6〜7割に抑え、息の成分をやや多めに混ぜる。「頭の中で鳴っている声」は、外に向かって発声された声とは根本的に質感が違う。共鳴は鼻腔寄りの柔らかい響きで、口腔を閉じ気味にして「声の輪郭をわずかにぼかす」。テンポは思考のスピードに従い、感情が高まると加速し、自分にブレーキをかけると停止する。

（顔が熱くなるのが分かる。本で顔を隠すように）

- 「今日も、来るかな。」：思考の起動。外界の音ではなく「頭の中で鳴っている期待」。声の輪郭をぼかし、空間に響かない——閉じた空間の中で鳴っている音。口を大きく開けず、鼻腔に声を通すことでこの質感を作る
- 「嘘。ぜんぜん偶然じゃない。」：自分への正直な告白。モノローグの自問は他者に投げかける疑問とは違い、答えを自分で知っている。テンポが一瞬止まり、「嘘」の一語で思考が方向転換する
- 「気持ち悪いかな。こんなの。」：自己否定。声のトーンが下がり、息の量が増える。内省の深い部分——自分の感情を持て余す苦しさが、息の重さに表れる
- 「来た。」：一語に心臓の音を込める。テンポが急停止し、呼吸が止まる。モノローグの中で最も「身体的な瞬間」——思考が身体反応に追い越される

- **全体**: モノローグの演技で最も重要なのは「声を誰にも聴かせていない」ということ。通常のセリフは「伝達」だが、モノローグは「自己対話」。声に方向性がなく、空間に拡散もしない。頭蓋骨の中で響いている——そのイメージを持つと、自然と声の出し方が変わる。恋するモノローグ特有の難しさは「感情が大きいのに声は小さい」こと。心臓がうるさいほどの感情を、内声の静けさの中に閉じ込める——この矛盾を声で表現できたとき、リスナーの胸を打つモノローグになる

## 応用① 距離がリアルタイムで変わるシーン（青春ドラマ）

> **目的**: 一つのセリフの中で距離が連続的に変化する。マイクとの物理的距離をリアルタイムで変えながら、声の質感を滑らかに遷移させる技術を鍛える。
> **キャラクター**: 18歳の高校三年生。卒業式の後、転校してしまう親友を校門で見送る。最初は隣に立って普通に話しているが、親友が歩き始め、距離が離れていくにつれ声量と距離感が変化する。最後は遠く離れた相手に叫ぶ。
> **やり方**: 台詞の進行に合わせ、マイクとの距離を段階的に変える。（通常会話15cm → やや離れて20cm → 中距離25cm → 遠距離30cm+）。声量・共鳴・息の配分も連動して変化させること。

（校門前。桜が舞う中、親友と並んで立っている。通常会話の距離）

【通常会話 / マイク15cm】
——元気でね。手紙、書くから。……ちゃんと返事くれなきゃ怒るからね。
（親友が歩き始める。数歩離れたところで振り返る。声を少し張る）

【やや遠い / マイク20cm】
あっ——マフラー！　あんたのマフラー、あたしがまだ持ってる！
（親友が笑って「あげる」と手を振る。さらに離れていく。声をもう一段張る）

【中距離 / マイク25cm】
いいのー！？　これ、お気に入りって言ってたじゃーん！
（親友の背中が小さくなっていく。桜並木の向こうに。叫ぶ）

【遠距離 / マイク30cm】
——ねぇっ！！
（親友が振り返る。最後の叫び。涙声で、でも笑って）
大好きだよー！！　——ずっとずっと、親友だからねー！！

### 距離変化のポイント
- **通常会話→やや遠い**: 声量を1.3倍に。共鳴ポイントを口腔から口腔＋鼻腔に広げる。テンポはやや速くなる（「マフラー」の焦りも加わるため自然に加速する）
- **やや遠い→中距離**: 声量を1.8倍に。フレーズの語尾を伸ばし始める（「言ってたじゃーん！」の「ーん」が距離を飛ばす機能を持つ）。マイクから離れるに従い、声の芯を太くする
- **中距離→遠距離**: 声量を最大に。全共鳴を解放。「大好きだよー！」は一語に全エネルギーを注ぎ込む。この「大好き」は会話の言葉ではなく、空間を貫通するために放たれた叫び
- **空間の変化を声で描く**: 校門前の桜並木は道幅があり、声が両側の木々に反射する。遠距離の声ほど語尾を「漂わせる」ことで、開けた空間の残響を暗示する
- **涙声の処理**: 遠距離で泣きながら叫ぶ場合、涙で声が震えるのは自然だが、声の芯まで崩れると「聞こえない叫び」になる。感情は全開、でも声の方向はブレない——泣きの叫びの技術はここにある

### 切り替えのポイント
- **電話→対面の最大の差は「声の拡散範囲」**: 電話は一方向に絞った声、対面は空間全体に広がる声。切り替えの瞬間に共鳴ポイントが変わる——電話の口腔中心から、対面の口腔＋鼻腔へ
- **テンポの変化**: 電話は母親への「元気な自分」を演じるテンポ。対面は同僚との自然な会話リズム。どちらも「本当の自分」ではない——電話の嘘と対面の社交、二種類の「仮面」を距離感で演じ分ける
- **声色の変化**: 電話で母親に話していた声（甘え混じりの娘の声）と、同僚に話す声（社会人としてのフランクな声）の二重の変化を一瞬で処理する。距離感の変化と人間関係の変化が同時に起こるため、高度な切り替えが求められる
- **ため息が「切り替えのブリッジ」**: 電話を切った後のため息が、電話モードから対面モードへの移行を示す。このため息に「仕事の疲れ」と「親に心配かけている罪悪感」が乗ると、キャラクターの内面が一瞬で伝わる

# 【収録ガイド】

## 推奨収録順

| 順番 | 内容 | 理由 |
|------|------|------|
| 1 | 通常会話①（放課後のおしゃべり） | 声のベースラインを確認。最も自然な距離感から始め、マイクとの標準距離を身体に馴染ませる |
| 2 | 電話越し①（親友との夜の電話） | 通常会話の距離感をベースに、声の方向性を絞る練習。声量の大幅な変更なしに質感だけを変える |
| 3 | モノローグ①（恋する女の子の心の声） | 声量を落とし、内声の質感を作る。声を「閉じる」技術を、喉が温まった状態で練習する |
| 4 | 至近距離①（親友への秘密話 × ASMR風） | 声帯の振動を最小限にする繊細な発声。喉のウォームアップが十分な状態で行うことで、囁きの中にも声の芯を保てる |
| 5 | 遠距離①（魔法少女の決め台詞） | 声量の最大値を出すシーンは最後に。喉が十分に温まり、腹式呼吸の準備が整った状態で収録する |
| 6 | 応用①（距離がリアルタイムで変わるシーン） | 5距離帯の個別練習が終わった後に、動的な距離変化に挑戦する |
| 7 | 応用②（同一セリフ・5距離帯読み分け） | 同じ台詞で距離感の振り幅を最大化するトレーニング |
| 8 | 応用③（電話→対面の切り替え） | 距離感と声質の同時切り替え。全技術の統合 |

## 録音チェックリスト

### 収録前
- [ ] マイクスタンドの高さを口の位置に合わせ、各距離帯でのポジション（3cm / 15cm / 25cm / 30cm）を事前にマーキングする
- [ ] ポップガード装着。特に至近距離では吹かれ防止が必須。遠距離の叫びでもポップ音が入りやすい
- [ ] 室内の環境ノイズを確認。至近距離の収録ではマイクゲインを上げるため、普段は拾わないノイズ（エアコン、PC冷却ファン等）が収録に乗る
- [ ] 水を用意（常温）。リップノイズ防止のため、口内を十分に湿らせる。リップクリームも準備
- [ ] 腹式呼吸のウォームアップ：仰向けに寝て腹に手を当て、3秒吸い→5秒吐きを10回。遠距離の叫びに備える
- [ ] 囁きの準備運動：「あいうえお」を吐息だけで発声し、声帯の振動を最小限にするコントロールを確認。サ行を入れた「さしすせそ」も追加し、歯擦音の柔らかさを確認
- [ ] 高音域の叫び準備：リップロールで喉をリラックスさせた後、ハミングで鼻腔共鳴を確認。高音の叫びで喉を締めないための事前チェック

### 収録中
- [ ] 各距離帯の冒頭に「テスト読み」を入れ、マイク距離とゲインの設定を確認してから本番に入る
- [ ] 至近距離の収録は、リップノイズ・ブレスノイズのレベルをモニタリングしながら行う。大きすぎれば口内を湿らせ、小さすぎればマイクにさらに近づく
- [ ] 至近距離で歯擦音（サ行・シャ行）が刺さっていないか確認。刺さっている場合はマイクの角度を微調整するか、息の方向をずらす
- [ ] 遠距離の叫びは、最低2テイクの間に1分以上の休憩を挟む。喉の疲労を感じたら即座に中断する。女性は特に高音の叫びで声帯を痛めやすいため、無理は厳禁
- [ ] 通常会話のテイクでは「演じている感」がないか特に注意。録音を即時再生し、不自然な箇所がないか確認
- [ ] 応用パートの距離変化シーンは、マイクとの距離を実際に動きながら収録する。足音や衣擦れのノイズが入らないよう、上半身のみで距離を調整する

### 収録後
- [ ] 5距離帯を連続再生し、「目を閉じて聴いた時に距離が見えるか」を確認する
- [ ] 至近距離と通常会話の差が出ているか。声量だけでなく、息の量・共鳴・テンポの変化が聴き取れるか
- [ ] 遠距離の叫びで声がクリップ（音割れ）していないか。波形を確認し、ピークが0dBを超えていたらマイク距離を再調整
- [ ] 遠距離の叫びで「キンキン」した不快な高音が出ていないか。出ている場合は胸腔の支えを追加する意識で再テイク
- [ ] 電話越しの声が、通常会話と「質感」として違って聞こえるか。声量が同じでも「電話だ」と分かるかがポイント
- [ ] モノローグが「小さな声で話しているだけ」になっていないか。「内声（頭の中の声）」と「小声（外に出ている声）」は違う。声の輪郭のぼかし方・共鳴の位置が変わっているか
- [ ] 応用①で距離の変化が滑らかに繋がっているか。急に声量が変わる不自然なポイントがないか。「グラデーション」として距離が変わっているか
- [ ] 苦手な距離帯を特定し、次回の重点練習項目としてメモする。多くの女性声優は「至近距離の歯擦音コントロール」と「遠距離で喉を締めない叫び」に課題を感じやすい

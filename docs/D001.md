# 『本を読み終えるまで』

## テ―マ

今回のお話はほぼ1シチュエ―ションで展開されます。舞台は古本屋、登場人物は基本的に二人です。そして、シ―ン1から6にかけて、二人の間になにか劇的な変化が訪れるわけではありません。しかし、緩やかに、緩やかに関係性は変化していきます。繊細な変化を演じるときに、見栄えのいいお芝居は助けになりません。変わらない日々に散りばめられた少しの変化と、その変化に自分がどう影響を受けたのかを丁寧に見つめてみて下さい。

## 登場人物

**徳井浩志（30）**  
古本屋の店主。本が好きで物知りだが、恋愛ベタで、学生時代にフラれた美桜を未だに引きずっており、最近その美桜が結婚するらしいと聞き、悶々としている。独り暮らし。店に来る野良猫に餌をやっている。

**北村唯（18）**  
女子高生。徳井のことが好きで、名前と電話番号を知ってもらおうとわざと万引きをする。徳井との結婚を望んでいるが、それより先に性的な関係を持ちたいとは思っていない。

**正雄（18）**  
唯の同級生。唯のことが好きで何度もアタックしているが、秒でフラレている。

**美桜（30）**  
学生時代、浩志とバイト先が同じだった。そのときに一度告白され断っているが、浩志が未だに自分に想いを寄せていることはなんとなく気づいている。婚約し、結婚に向けて準備をしている最中、同居している婚約者の浮気が発覚し、「私も誰かと一度だけ寝てくる！」と家を飛び出し、浩志に連絡をした。

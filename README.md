# Voice Actor Laboratory

声優が台本を見ながら複数テイクを録音・再生・比較するデスクトップアプリです。

## 必要な環境

- Python 3.10+
- マイク（録音時）

## セットアップ

```bash
pip install -r requirements.txt
```

## 起動

```bash
python main.py
```

## 初めての使い方（クイックスタート）

初めて起動したときは「はじめに」画面が出ます。**録音 → 再生**まで次の順番で進めます。

| 手順 | やること |
|------|----------|
| 1 | **「新規プロジェクトを作成」** をクリックし、保存先のフォルダを選ぶ |
| 2 | 台本エリアに例が表示されるので、そのまま使うか編集する |
| 3 | **「録音開始」**（F9）で録音 → **「録音停止」**で 1 テイク追加 |
| 4 | 右側の **テイク一覧でダブルクリック** すると再生される |

※ いつでも **ヘルプ → 使い方** で手順を確認できます。

---

## 使い方（詳細）

1. **新規プロジェクト** … 「ファイル」→「新規プロジェクト」で保存先フォルダを選ぶ
2. **台本を開く** … 「ファイル」→「台本を開く」で .txt を選択（任意）
3. **録音** … 「録音開始」→「録音停止」で 1 テイクがプロジェクトに追加される
4. **再生** … テイク一覧でダブルクリックで再生
5. **お気に入り** … テイクを右クリック→「お気に入りに追加/解除」
6. **エクスポート** … 「ファイル」→「テイクをエクスポート」で WAV を指定フォルダにコピー
7. **検索・置換** … 「編集」→「検索」（Ctrl+F）／「置換」（Ctrl+H）で台本内を検索・置換

## 台本フォーマット

- **見出し**: 行頭 `# ` / `## ` / `### ` の行はシーン・カット名。録音ファイル名が「見出し_001.wav」のように付く。
- **セリフ**: `キャラ名: セリフ` の形式で書くと、個別モード時にセリフ部分がテイク名などに反映される。
- **録音モード**: 「台本一括」（bulk）は見出しごとに連番、「セリフ個別」（individual）はカーソル行のセリフをファイル名に使います。ヘルプ→使い方でも確認できます。
- テキストは UTF-8 で保存。読み込み時は UTF-8（BOM あり/なし）・CP932 に自動フォールバックする。

## 既知の制限・注意事項

- 同時に開けるプロジェクトは 1 つです。別のプロジェクトを開くと現在のプロジェクトが切り替わります。
- 録音は最大約 3 時間まで（メモリ保護のため）。それ以上はバッファに追加されません。
- 波形表示は約 10 分以上の音声は末尾 10 分のみ表示します。

## テスト

```bash
python -m pytest tests/ -v
```

カバレッジを確認する場合（`pip install pytest-cov` が必要）:

```bash
python -m pytest tests/ -v --cov=src --cov-report=term-missing
```

## 構成

- `src/project.py` … プロジェクト・台本・テイクメタ
- `src/storage.py` … プロジェクト保存/読み込み/エクスポート
- `src/recorder.py` … 録音（sounddevice）
- `src/playback.py` … 再生（PyQt6 QMediaPlayer）
- `src/ui/main_window.py` … メインウィンドウ
